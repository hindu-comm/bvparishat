As a convention, the underbar is used for alveolar consonants and a dot below is used for retroflex consonants. Therefore,ISO 15919 reserves ṛ
for ड़ (ড় ੜ ଡ଼), andṛh for ढ़. Dravidian linguistic literature usesḏ for voiced alveolar stops.

  

Retroflex:

|     |     |                                    | |:---:|-----|------------------------------------| |  ṭ  | ट   | voiceless retroflex stop           | | ṭh  | ठ   | voiceless aspirated retroflex stop | |  ḍ  | ड   | voiced retroflex stop              | | ḍh  | ढ   | voiced aspirated retroflex stop    | |  ṛ  | ड़  | retroflex flap                     | | ṛh  | ढ़  | aspirated retroflex flap           | |  ṇ  | ण   | retroflex nasal stop               | |  ḷ  | ळ   | retroflex lateral                  |
