     + Structural semantics  
     + Machine Translation  
     + Automatic analysis ofSanskritcorpus  
     + Machine Learning approaches tocomputationalprocessing  
     + Navya NyÄya technical language processing and semantic analysis  
     + Information extraction  
  
   o ShāstricSanskrittexts and computation  
     + Computer modelling and simulation of Paninian and other  
      traditional grammars  
     + Theories of śābdabodha andSanskritcomputationalprocessing  
     +Sanskritdigitallibraries management: Tools for acquisition
