In Bucknell's Sanskrit Manual, on page 191, root 320 vac-2, in FPP (there are 2 of them in each dhatu entry) there is a suspicious form vācanīya. In <http://www.warnemyr.com/skrgram/grammar/roots/root_vac.html> in Derivatives both are there vacanīya S.+ and -vācanīya C.  

I would expect the form to be vacanīya, the corpora agrees <http://kjc-sv013.kjc.uni-heidelberg.de/dcs/index.php?contents=fundstellen&IDVerbalform=38149>

>   
> Rāmāyaṇa  
> Rām, 7, 46, 12.1 na tāni vacanīyāni mayā devi tavāgrataḥ /  
> Kirātārjunīya  
> Kir, 9, 39.1 ucyatāṃ sa vacanīyam aśeṣaṃ neśvare paruṣatā sakhi sādhvī > /  
> Kumārasaṃbhava  
> KumSaṃ, 4, 21.2 vacanīyam idaṃ vyavasthitaṃ ramaṇa tvām anuyāmi yady > api //  
> KumSaṃ, 5, 82.2 mamātra bhāvaikarasaṃ manaḥ sthitaṃ na kāmavṛttir > vacanīyam īkṣate //  
> Āryāsaptaśatī  
> Āryāsaptaśatī, 2, 362.2 capalāntara ghana kiṃ tava vacanīyaṃ > pavanavaśyo 'si //

  

So the question remains open - should the main form be vacanīya instead of vācanīya or are both possible and there should be no preference? vācanīya vs. vacanīya.
