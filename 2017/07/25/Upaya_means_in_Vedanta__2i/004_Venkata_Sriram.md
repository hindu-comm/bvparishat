Namaste,

  

In Raja Yoga, there is a path called 'tAraka-yoga' which is explained in 'maNDala-brAhmaNa-vidya'. This secret doctrine is explained inमण्डल ब्राह्मण उपनिषद् for which there is a wonderful sanskrit commentary byसदानन्द अवधूत. I have read this completely. The book can be downloaded here:

  

...

The सदानन्द अवधूत comments as:

  

ननु अन्यशास्त्राणि संख्यादीनि बहूनि विद्यन्ते, तानि च तत्वप्रधानान्येव , तद्वारा ब्रह्मप्राप्तिरस्तु किमेतदुपदेशेन ? तत् सर्व प्रयत्नेन योगाभ्यास एव ब्रह्मप्राप्ततये कर्तव्यः

  

The yoga-mArga is the best upAsya for brahma-prApta - says Sadananda Avadhuta. Then what is this yoga-mArga? 

  

Hence, this tAraka-brahma-yoga (maNDala-vidya).
