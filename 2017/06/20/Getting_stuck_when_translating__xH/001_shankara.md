The following version of this verse (copied from a web page) seems relatively free from errors.  

  

तापिञ्छश्यामळाँगस्मितमधुरमुखं पिञ्छभारावतंसः  
प्रौन्मीलद्यौवनश्रीररुणवसनभृत्कर्णिकाकर्णिकारः।  
वेणुक्वाणैरुदारैर्व्रजयुवतीमनक्षोभमापादयन्तो  
वश्यं निश्शेषविश्वं कलयति यमुनाकूलवासी मुकुन्दः॥



regards  
shankara
