तिङन्तम्। “इण् गतौ”इति धातुः।

  

“उपेयिवाननाश्वाननूचानश्च”(३.२.१०९) इति सूत्रे सिद्धान्तकौमुदी:“उपेत्यविवक्षितम्। ईयिवान्। समीयिवान्।”

  

एवमेव“आ +ईयिवान् =एयिवान्।”एयिवान् = आगात् = ऐत् = एयाय।

  

आ + इण् गतौ → भूतकालविवक्षा → आ + इण् + लिट्/लुङ्/लङ् → उपेयिवाननाश्वाननूचानश्च (पा॰सू॰३.२.१०९) → आ + इण् + क्वसुँ → अनुबन्धलोपः → आ इ वस् → लिटि धातोरनभ्यासस्य (पा॰सू॰६.१.८) → आ इ इ वस् → दीर्घ इणः किति (पा॰सू॰७.४.६९) → आ ई इ वस् → विधानसामर्थ्यात्सवर्णदीर्घप्रतिषेधः → अनेकाच्त्वात्क्वसोरिडागमाप्राप्तौ प्रकृतसूत्रेणेट् निपात्यते → आ ई इ इट् वस् → अनुबन्धलोपः → आ ई इ इ वस् → इणो यण् (पा॰सू॰६.४.८१) → आ ई य् इ वस् → आ ईयिवस् → आद्गुणः (पा॰सू॰\~६.१.८७) → एयिवस् → विभक्तिकार्यम् → एयिवान्।
