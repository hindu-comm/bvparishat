It does not, the form in लुङ् 2^(nd) plural is अयासिष्ट with अ at beginning, due to लुङ्लङ्लृङ्क्ष्वडुदात्तः (पा॰सू॰ ६.४.७१). The form in RV is यासीष्ट without the अ and with the long vowel.

या प्रापणे is अदादि परस्मैपदी अनिट्. But here the usage is in आत्मनेपद due to व्यत्ययो बहुलम् (पा॰सू॰ ३.१.८५). This is not ungrammatical, but Vedic usage. The derivation is as follows:

  

या प्रापणे → आशिषि लिङ्लोटौ (पा॰सू॰ ३.३.१७३) → या लिङ् → शेषात्कर्तरि परस्मैपदम् (पा॰सू॰ १.३.७८) → व्यत्ययो बहुलम् (पा॰सू॰ ३.१.८५) → वेदे व्यत्ययेनात्मनेपदप्राप्तिः → या लिङ् → तिप्तस्झिसिप्थस्थमिब्वस्मस्ताताञ्झथासाथान्ध्वमिड्वहिमहिङ् (पा॰सू॰ ३.४.७८) → या त → लिङः सीयुट् (पा॰सू॰ ३.४.१०२) → या सीयुँट् त → सुँट् तिथोः (पा॰सू॰ ३.४.१०७) → या सीयुँट् सुँट् त → अनुबन्धलोपः → या सीय् स् त → लोपो व्योर्वलि (पा॰सू॰ ६.१.६६) → या सी स् त → आदेशप्रत्यययोः (पा॰सू॰ ८.३.५९) → या सी ष् त → ष्टुना ष्टुः (पा॰सू॰ ८.४.४१) → या सी ष् ट → यासीष्ट
