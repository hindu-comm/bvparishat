.db files are database files created in popular lightweight database SQLite.

## MacOS X

DB Browser for SQLite works well on MacOS X.

· OSX 10.8 (Mountain Lion) - 10.11 (El Capitan) are tested and known to work

### MacOS X binaries

OSX binaries can be downloaded from here:

·
[https://github.com/sqlitebrowser/sqlitebrowser/releases](https://github.com/sqlitebrowser/sqlitebrowser/releases)

Latest OSX binary can be installed via[Homebrew Cask](https://caskroom.github.io/):

 `brew cask install sqlitebrowser`

Nightly builds for OSX are available at:

·
[https://nightlies.sqlitebrowser.org/osx/](https://nightlies.sqlitebrowser.org/osx/)
