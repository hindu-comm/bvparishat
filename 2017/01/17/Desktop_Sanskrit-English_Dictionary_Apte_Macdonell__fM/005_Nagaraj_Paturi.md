.db is a database file.

  

.db files are database files created in popular lightweight database SQLite.  

  

DB Browsre for SQ Lite is a software that can help you create browse and edit .db files.

  

Your purpose is to browse.

  

## MacOS X

DB Browser for SQLite works well on MacOS X.

-   OSX 10.8 (Mountain Lion) - 10.11 (El Capitan) are tested and known
    to work

### [](http://sqlitebrowser.org/#macosx-binaries)MacOS X binaries

OSX binaries can be downloaded from here:

-   <https://github.com/sqlitebrowser/sqlitebrowser/releases>

Latest OSX binary can be installed via[Homebrew Cask](https://caskroom.github.io/):

 `brew cask install sqlitebrowser`

Nightly builds for OSX are available at:

-   <https://nightlies.sqlitebrowser.org/osx/>
