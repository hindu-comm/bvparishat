प्रभवो विभवः शुक्लः प्रमोदोथ प्रजापतिः

अंगिराः श्रीमुखो भावो युवा धाता तथेश्वरः //

बहुधान्यः प्रमाथी च विक्रमो वृषसंज्ञकः

चित्रभानुः सुभानुश्च तारणः पार्थिवो व्ययः //

सर्वजित सर्वधारी च विरोधी विकृतः खरः

नंदनो विजयश्चैव जयो मन्मथदुर्मुखौ //

हेमलंबो विलंबश्च विकारीशार्वरी प्लवः

शुभकृच्छोभनः क्रोधी विश्वावसुपराभवौ //

प्ल्वंगः कीलकः सौम्यः साधरणो विरोधकृत्

परीधावी प्रमादी च आनंदो राक्षसोनलः//

पिंगलः कालयुक्तश्च सिद्धार्थी रौद्रदुर्मती

दुंदुभी रुधिरोद्गारी रक्ताक्षी क्रोधनः क्षयः//

///////////////////////////////////  

  

regs,

Sriram
