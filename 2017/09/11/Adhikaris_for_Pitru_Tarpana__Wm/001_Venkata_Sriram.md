Namaste,

  

Theनित्यपितृ तर्पणं ofब्रह्म यज्ञं should be performed by all, irrespective of whether one's father is alive or not. I do ब्रह्म यज्ञं daily which consists ofस्वाध्याय of one'sशाख (a singleऋक् is taken fromसंहिता, ब्राह्मण, आराण्यक),गायत्री मन्त्रं (व्याहृतित्रय सहितं) etc. The second part of Brahma Yajnam consists ofदेव, ऋषि, पितृ तर्पणं and separate samkalpa is done as a part of Brahma Yajna. देव तर्पणं is done once withउपवीति;ऋषि तर्पणं is done twice withनिवीति andपितृ तर्पणं is done thrice withप्राचीनवीति. Regarding theप्राचीनवीति position w.r.tजीव पितरः, there are regional differences. However,पितृ तर्पणं as perआपस्तंब is given below which I follow every day.
Theतर्पणं should be given with water and NOT with sesame seeds.

सोमः पितृमान् यमोऽङ्गिरस्वान् अग्निकव्यवाहनादयः ये पितरः तान् पितृगं् तर्पयामि (त्रिवारम्)

सर्वान् पितृगं् तर्पयामि (त्रिवारम्)

सर्वान् पितृगणागं् तर्पयामि (त्रिवारम्)

सर्वाः पितृपत्नीस्तर्पयामि (त्रिवारम्)

सर्वान् पितृपुत्रागं् तर्पयामि (त्रिवारम्)

सर्वान् पितृपौत्रागं् तर्पयामि (त्रिवारम्)

भूः पितृगं् तर्पयामि (त्रिवारम्)

भुवः पितृगं् तर्पयामि (त्रिवारम्)

सुवः पितृगं् तर्पयामि (त्रिवारम्)

भूर्भुस्सुवः पितृगं् तर्पयामि (त्रिवारम्)
