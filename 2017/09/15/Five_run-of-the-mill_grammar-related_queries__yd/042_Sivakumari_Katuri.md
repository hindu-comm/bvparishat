> 
> > Second and more importantly, this line from Bhaimi draws forth the > main substratum of this differentiation; i.e. when there is a > prayojana it is an ‘it’; otherwise not. >
> 

तर्हि सु इति प्रथमैकवचनप्रत्यये उकारस्य इत्संज्ञा अस्ति न वा, यद्यस्ति तस्य किं प्रयोजनम्।
