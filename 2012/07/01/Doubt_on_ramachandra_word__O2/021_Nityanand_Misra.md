This is not an example of रामचन्द्र name but the रूपक रामचन्द्रमस् (as you have the रूपक of रावणराहु in the same line). If it were were an example of the former, the singular accusative form would be रामचन्द्रं and not रामचन्द्रमसं. रामचन्द्रमस् is not used as a name but as a metaphor in this verse.  
  
Thanks for the reference anyway - some minor corrections are highlighted below:  
  
विषेदुर्देवगन्धर्वा दानवाश्चारणैः सह ।। ६.१०३.२७ ।।  
राममार्तं तदा दृष्ट्वा सिद्धाश्च परमर्षयः ।  
व्यथिता वानरेन्द्राश्च बभूवुः सविभीषणाः ।। ६.१०३.२८ ।।  
रामचन्द्रमसं दृष्ट्वा ग्रस्तं रावणराहुणा ।  
प्राजापत्यं च नक्षत्रं रोहिणीं शशिनः प्रियाम् ।। ६.१०३.२९ ।।  
  
First two instances can have न् if there is external Sandhi in which case the words would have to be typed without a gap in keeping with the punctuation convention.
