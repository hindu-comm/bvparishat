Dear Dr. Ramanujan  
  
Thanks again for pointing out other विद् roots. I checked again, and there are in fact five different विद् roots in five different गणs. I also derived the लृङ् forms (see below) for all and from this exercise I do not see वेत्स्यति/अवेत्स्यत् coming from विदँ ज्ञाने in Paninian system - from what I see it can only come from विद्ऌँ लाभे - and indeed it is listed under विद्ऌँ लाभे in both बृहद्धातुरूपावली as well as धातुरत्नाकर. It may have been used in the sense in Mahabharatam and Puranas - which appears to be an अपाणिनीय प्रयोग. I have no idea why Whitney included वेत्स्यति under विदँ ज्ञाने. Maybe he grammarians can throw some light here.

विदँ ज्ञाने – परस्मैपदे लृँङि प्रथमपुरुष एकवचने

विद् -> लिँङ्‌निमित्ते लृँङ् क्रियाऽतिपत्तौ (३-३-१३९) -> विद लृँङ् -> विद् ल -> तिप्तस्झिसिप्थस्थमिब्वस्मस्तातांझथासाथांध्वमिड्वहिमहिङ् (३-४-७८) -> विद् तिप् -> विद् ति -> इतश्च (३-४-१००) -> इलोपः -> विद् त् -> स्यतासी लृलुटोः (३-१-३३) -> स्यप्रत्ययः -> विद् स्य त् -> स्यस्य आर्धधातुकसंज्ञा -> विकरणबाधनम् -> आर्धधातुकस्येड्वलादेः (७-२-३५) -> इडागमः -> आद्यन्तौ टकितौ (१-१-४६) -> आद्यागमः -> विद् इट् स्य त् -> विद् इ स्य त् -> पुगन्तलघूपधस्य च (७-३-८६) -> लघूपधगुणः -> अदेङ्गुणः (१-१-२) -> स्थानेऽन्तरतमः (१-१-५०) -> एगुणः -> वेद् इ स्य त् -> लुङ्लङ्लृङ्क्ष्वडुदात्तः (६-४-७१) -> अडागमः -> आद्यन्तौ टकितौ (१-१-४६) -> आद्यागमः -> अट् वेद् इ स्य त् -> अ वेद् इ स्य त् -> अवेदिष्यत्



विदँ सत्तायाम्, विद्ऌँ लाभे, विदँ विचारणे – आत्मनेपदे लृँङि प्रथमपुरुष एकवचने

विद् -> लिँङ्‌निमित्ते लृँङ् क्रियाऽतिपत्तौ (३-३-१३९) -> विद लृँङ् -> विद् ल -> तिप्तस्झिसिप्थस्थमिब्वस्मस्तातांझथासाथांध्वमिड्वहिमहिङ् (३-४-७८) -> विद् त -> स्यतासी लृलुटोः (३-१-३३) -> स्यप्रत्ययः -> विद् स्य त -> स्यस्य आर्धधातुकसंज्ञा -> विकरणबाधनम् -> विद् स्य त -> एकाच उपदेशेऽनुदात्तात्‌ (७-२-१०) -> इण्निषेधः -> विद् स्य त -> पुगन्तलघूपधस्य च (७-३-८६) -> लघूपधगुणः -> अदेङ्गुणः (१-१-२) -> स्थानेऽन्तरतमः (१-१-५०) -> एगुणः -> वेद् स्य त -> लुङ्लङ्लृङ्क्ष्वडुदात्तः (६-४-७१) -> अडागमः -> आद्यन्तौ टकितौ (१-१-४६) -> आद्यागमः -> अट् वेद् स्य त -> अ वेद् स्य त -> खरि च (८-४-५५) -> चर्त्वम् -> अ वेत् स्य त् -> अवेत्स्यत



विद्ऌँ लाभे – परस्मैपदे लृँङि प्रथमपुरुष एकवचने

विद् -> लिँङ्‌निमित्ते लृँङ् क्रियाऽतिपत्तौ (३-३-१३९) -> विद लृँङ् -> विद् ल -> तिप्तस्झिसिप्थस्थमिब्वस्मस्तातांझथासाथांध्वमिड्वहिमहिङ् (३-४-७८) -> विद् तिप् -> विद् ति -> इतश्च (३-४-१००) -> इलोपः -> विद् त् -> स्यतासी लृलुटोः (३-१-३३) -> स्यप्रत्ययः -> विद् स्य त् -> स्यस्य आर्धधातुकसंज्ञा -> विकरणबाधनम् -> एकाच उपदेशेऽनुदात्तात्‌ (७-२-१०) -> इण्निषेधः -> विद् स्य त् -> पुगन्तलघूपधस्य च (७-३-८६) -> लघूपधगुणः -> अदेङ्गुणः (१-१-२) -> स्थानेऽन्तरतमः (१-१-५०) -> एगुणः -> वेद् स्य त् -> लुङ्लङ्लृङ्क्ष्वडुदात्तः (६-४-७१) -> अडागमः -> आद्यन्तौ टकितौ (१-१-४६) -> आद्यागमः -> अट् वेद् स्य त् -> अ वेद् स्य त् -> खरि च (८-४-५५) -> चर्त्वम् -> अ वेत् स्य त् -> अवेत्स्यत्

  

विदँ चेतनाखयाननिवासनेषु – आत्मनेपदे लृँङि प्रथमपुरुष एकवचने

विद् -> सत्यापपाशरूपवीणातूलश्लोकसेनालोमत्वचवर्मवर्णचूर्णचुरादिभ्यो णिच् (३-१-२५) -> विद् णिच् -> विद् इ -> पुगन्तलघूपधस्य च (७-३-८६) -> लघूपधगुणः -> अदेङ्गुणः (१-१-२) -> स्थानेऽन्तरतमः (१-१-५०) -> एगुणः -> वेद् इ -> वेदि -> लिँङ्‌निमित्ते लृँङ् क्रियाऽतिपत्तौ (३-३-१३९) -> वेदि लृँङ् -> वेदि ल -> तिप्तस्झिसिप्थस्थमिब्वस्मस्तातांझथासाथांध्वमिड्वहिमहिङ् (३-४-७८) -> वेदि त -> स्यतासी लृलुटोः (३-१-३३) -> स्यप्रत्ययः -> वेदि स्य त -> स्यस्य आर्धधातुकसंज्ञा -> विकरणबाधनम् -> सार्वधातुकार्धधातुकयोः (७-३-८४) -> अङ्गगुणः -> अदेङ्गुणः (१-१-२) -> स्थानेऽन्तरतमः (१-१-५०) -> एगुणः -> वेदे स्य त -> आर्धधातुकस्येड्वलादेः (७-२-३५) -> इडागमः -> आद्यन्तौ टकितौ (१-१-४६) -> आद्यागमः -> वेदे इट् स्य त -> वेदे इ स्य त -> एचोऽयवायावः (६-१-७८) -> यथासंख्यमनुदेशः समानाम् (१-३-१०) -> अयादेशः -> वेद् अय् इ स्य त -> लुङ्लङ्लृङ्क्ष्वडुदात्तः (६-४-७१) -> अडागमः -> आद्यन्तौ टकितौ (१-१-४६) -> आद्यागमः -> अट् वेद् अय् इ स्य त -> अ वेद् अय् इ स्य त -> अवेदयिष्यत
