> > As far as my limited knowledge goes, रङ्ग cannot come from भावे घञ् or > करणे घञ् in the Paninian system, but it can only अधिकरणे घञ्. Please > see  
>   
> 6.4.27 घञि च भावकरणयोः \[अङ्गस्य उपधायाः रञ्जेः नलोपः\]  
>   
> As per this rule, for भावे घञ् (रञ्जनम्) or करणे घञ् (रजति/रज्यति > अनेन) the procedure is  
>   
> रञ्ज् + घञ् -> घञि च भावकरणयोः (6.4.27) -> रज् + घञ् -> अनुबन्धलोप -> > रज् + अ -> अत उपधायाः (7.2.116) -> राज् घञ् -> चजोः कु घिण्ण्यतोः > (7.3.53) -> कुत्व -> गुणकृत आन्तर्य -> गत्व -> राग् अ -> राग -> राग > सु -> रागः  
>   
> If it is अधिकरणे घञ् (रजति/रज्यति अस्मिन्) as Dr. Bhat has pointed > out, the procedure is  
>   
> रञ्ज् + घञ् -> अनुबन्धलोप -> रञ्ज् + अ -> चजोः कु घिण्ण्यतोः (7.3.53) > -> कुत्व -> गुणकृत आन्तर्य -> गत्व -> रञ् ग अ -> नश्चापदान्तस्य झलि > (8.3.24) -> रं ग् अ -> अनुस्वारस्य ययि परसवर्णः (8.4.57) -> रङ् ग् अ > -> रङ्ग -> रङ्ग सु -> रङ्गः  
>   
> PS: Note that the उपधा in roots like रञ्ज् and गुम्फ् is originally न् > but becomes a different पञ्चमवर्ण due to 8.3.24 and 8.4.57. Which is > why Panini says रञ्जेः नलोपः which implies the उपधा in रञ्ज् is > actually न् and also we get गुम्फित्वा from नोपधात्थफान्ताद्वा > (1-2-23) which implies the उपधा in गुम्फ् is also न्.
