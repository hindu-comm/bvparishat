Yes, I am also using one 24" external monitor with my 14" Windows laptop. While doing transliteration or proof reading, I use three windows on these two screens at "comfortable" size simultaneously. If I need to open more windows, I keep them in the background and swap as and when I need to focus on any specific window.

  

After converting the manuscripts to text form, as Vasant ji mentioned, using Excel tool will be more useful - during critical editing stage. 

  

For using one external monitor no additional resources are required. One can easily plug the monitor to laptop's HDMI port. Tech savvy people can use even 3 or 4 external monitors. Windows operating system supports them. However, a strong processor, large RAM and few additional adopters may be required to support more than one external monitor.
