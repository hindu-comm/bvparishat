For convenience of others, pasting below the karikas 78-95 of काव्यादर्श of दण्डी as suggested by yoginiji:

  

वर्णानाम् एकरूपत्वं यद्य् एकान्तरम् अर्धयोः /  
गोमूत्रिकेति तत् प्राहुर् दुष्करं तद्विदो यथा // KDद्\_३.७८ //  
  
मदनो मदिराक्षीणाम् अपाङ्गास्त्रो जयेद् अयम् /  
मदेनो यदि तत् क्षीणम् अनङ्गायाञ्जलिं दधे // KDद्\_३.७९ //  
  
आहुर् अर्धभ्रमं नाम श्लोकार्धभ्रमणं यदि /  
तद् इष्टं सर्वतोभद्रं भ्रमणं यदि सर्वतः // KDद्\_३.८० //  
  
मनोभव तवानीकं नोदयाय न मानिनी /  
भयाद् अमेयामा मा वा वयम् एनोमया नत // KDद्\_३.८१ //  
  
सामायामा मा या मासा मारानायायाना रामा /  
यानावारारावानाया मायारामा मारायामा // KDद्\_३.८२ //  
  
यः स्वरस्थानवर्णानां नियमो दुष्करेष्व् असौ /  
इष्टश् चतुःप्रभृत्य् एष दर्श्यते सुकरः परः // KDद्\_३.८३ //  
  
आम्नायानाम् आहान्त्या वाग् ईतीर् गीतीर् भीतीः प्रीतीः /  
भोगो रोगो मोदो मोहो धेये धेच्छे देशे क्षेमे // KDद्\_३.८४ //  
  
क्षितिविजितिस्थितिविहितिव्रतरतयः परगतयः /  
उरु रुरुधुर् गुरु दुधुवुर् युधि कुरवः स्वम् अरिकुलम् // KDद्\_३.८५ //  
  
श्रीदीप्ती ह्रीकीर्ती धीनीती गीःप्रीती /  
एधेते द्वे द्वे ते ये नेमे देवेशे // KDद्\_३.८६ //  
  
सामायामा मा या मासा मारानायायाना रामा /  
यानावारारावानाया मायारामा मारायामा // KDद्\_३.८७ //  
  
नयनानन्दजनने नक्षत्रगणशालिनि /  
अघने गगने दृष्टिर् अङ्गने दीयतां सकृत् // KDद्\_३.८८ //  
  
अलिनीलालकलतं कं न हन्ति घनस्तनि /  
आननं नलिनच्छायनयनं शशिकान्ति ते // KDद्\_३.८९ //  
  
अनङ्गलङ्घनालग्ननानातङ्का सदङ्गना /  
सदानघ सदानन्द नताङ्गासङ्गसंगत // KDद्\_३.९० //  
  
अगा गां गाङ्गकाकाकगाहकाघककाकहा /  
अहाहाङ्ग खगाङ्कागकङ्कागखगकाकक // KDद्\_३.९१ //  
  
रे रे रोरूरुरूरोरुगागोगो ऽगाङ्गगो ऽगगुः /  
किं केकाकाकुकः काको मा मा माम् आम मामम // KDद्\_३.९२ //  
  
देवानां नन्दनो देवो नोदनो वेदनिन्दिनः /  
दिवं दुदाव नादेन दाने दानवनन्दिनः // KDद्\_३.९३ //  
  
सूरिः सुरासुरासारिसारः सारासिसारसाः /  
ससार सरसीः सीरी ससूरुः स सुरारसी // KDद्\_३.९४ //  
  
नूनं नुन्नानि नानेन नाननेनाननानि नः /  
नानेना ननु नानूनेनैनेनानान् इनो निनीः // KDद्\_३.९५ //
