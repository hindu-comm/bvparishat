\<\<I wanted to know from other Vaiyākaraṇas if there is any evidence for the root चुर् in the first conjugation.>\>

  

Vopadeva's Kavikalpadruma prescribes it as both bhvaadi and curaadi by using the marker "-ki" (pl. see the attached page, shloka 265).
