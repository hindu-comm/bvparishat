This is known as

*avimr̥ṣṭa-vidheyāṁśa.*

*uddeśya (*also called *anuvādya)* must appear first in a sentence.

*vidheya* must come only later.

If the order is reversed, it is a *doṣa*.

It is this *doṣa* that is referred to here.
