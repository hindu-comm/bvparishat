Here is the complete derivation of all the six forms listed by Puspa Diksita for वृञ् वरणे (वर्वर्ति, वर्वरीति, वरिवर्ति, वरिवरीति, वरीवर्ति, वरीवरीति).  

वृञ् वरणे (धा॰पा॰ १२५४) → वृ → धातोरेकाचो हलादेः क्रियासमभिहारे यङ् (पा॰सू॰ ३.१.२२) → वृ यङ् → यङोऽचि च (पा॰सू॰ २.४.७४) → चकारादच्प्रत्ययं विनाऽपि यङो लुक् → वृ → प्रत्ययलोपे प्रत्ययलक्षणम् (पा॰सू॰ १.१.६२) → सन्यङोः (पा॰सू॰ ६.१.९) → अभ्यासकार्यः → वृ वृ → उरत्‌ (पा॰सू॰\~७.४.६६) → व वृ → उरण् रपरः (पा॰सू॰ १.१.५१) → वर् वृ → हलादिः शेषः (पा॰सू॰ ७.४.६०) → व वृ → चर्करीतं च (धा॰पा॰ग॰सू॰ १०८१) → अदादिगणत्वेन धातुसञ्ज्ञा → वर्तमाने लट् (पा॰सू॰ ३.२.१२३) → शेषात्कर्तरि परस्मैपदम् (पा॰सू॰ १.३.७८) → व वृ तिप् → व वृ ति → कर्तरि शप् (पा॰सू॰ ३.१.६८) → व वृ शप् ति → अदिप्रभृतिभ्यः शपः (पा॰सू॰ २.४.७२) → शब्लुक् → व वृ ति → ऋतश्च (पा॰सू॰ ७.४.९२) → अभ्यासस्य पर्यायेण रुग्रिग्रीगागमाः।  
  
(क) रुक्पक्षे → आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → व रुँक् वृ ति → व र् वृ ति → यङो वा (पा॰सू॰ ७.३.९४) → सार्वधातुकस्य पितः तिपो वैकल्पिक ईडागमः।  
  
 (१) ईट्पक्षे → आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → व र् वृ ईट् ति → व र् वृ ई ति → सार्वधातुकार्धधातुकयोः (पा॰सू॰\~७.३.८४) → व र् व ई ति → उरण् रपरः (पा॰सू॰ १.१.५१) → व र् व र् ई ति → वर्वरीति।  
  
 (२) ईडभावे → व र् वृ ति → सार्वधातुकार्धधातुकयोः (पा॰सू॰\~७.३.८४) → व र् व ति → उरण् रपरः (पा॰सू॰ १.१.५१) → व र् वर् ति → वर्वर्ति। अचो रहाभ्यां द्वे (पा॰सू॰ ८.४.४६) इत्यनेन तकारस्य वैकल्पिकद्वित्वे वर्वर्त्ति इत्यपि।  
  
(ख) रिक्पक्षे → आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → व रिक् वृ ति → व रि वृ ति → यङो वा (पा॰सू॰ ७.३.९४) → सार्वधातुकस्य पितः तिपो वैकल्पिक ईडागमः।  
  
 (३) ईट्पक्षे → आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → व रि वृ ईट् ति → व रि वृ ई ति → सार्वधातुकार्धधातुकयोः (पा॰सू॰\~७.३.८४) → व रि व ई ति → उरण् रपरः (पा॰सू॰ १.१.५१) → व रि व र् ई ति → वरिवरीति।  
  
 (४) ईडभावे → व रि वृ ति → सार्वधातुकार्धधातुकयोः (पा॰सू॰\~७.३.८४) → व रि व ति → उरण् रपरः (पा॰सू॰ १.१.५१) → व रि वर् ति → वरिवर्ति। अचो रहाभ्यां द्वे (पा॰सू॰ ८.४.४६) इत्यनेन तकारस्य वैकल्पिकद्वित्वे वरिवर्त्ति इत्यपि।  
  
(ग) रीक्पक्षे → आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → व रीक् वृ ति → व री वृ ति → यङो वा (पा॰सू॰ ७.३.९४) → सार्वधातुकस्य पितः तिपो वैकल्पिक ईडागमः।  
  
  (५) ईट्पक्षे → आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → व री वृ ईट् ति → व री वृ ई ति → सार्वधातुकार्धधातुकयोः (पा॰सू॰\~७.३.८४) → व री व ई ति → उरण् रपरः (पा॰सू॰ १.१.५१) → व री व र् ई ति → वरीवरीति।  
  
  (६) ईडभावे → व री वृ ति → सार्वधातुकार्धधातुकयोः (पा॰सू॰\~७.३.८४) → व री व ति → उरण् रपरः (पा॰सू॰ १.१.५१) → व री वर् ति → वरीवर्ति। अचो रहाभ्यां द्वे (पा॰सू॰ ८.४.४६) इत्यनेन तकारस्य वैकल्पिकद्वित्वे वरीवर्त्ति इत्यपि।
