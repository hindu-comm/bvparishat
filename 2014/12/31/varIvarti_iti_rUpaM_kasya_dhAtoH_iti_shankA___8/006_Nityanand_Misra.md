Here is the complete derivation of all the six forms listed by Dr. Bhat (वर्वृतीति/वरिवृतीति/वरीवृतीति/वर्वर्ति/वरिवर्ति/वरीवर्ति). It was only today I got time for this. Please excuse any typos.  

वृतुँ वर्तने (धा॰पा॰ ७५८)/वृतुँ वरणे (धा॰पा॰ ११६०)/वृतुँ भाषार्थः (धा॰पा॰ १७८१) → वृतुँ → वृत् → धातोरेकाचो हलादेः क्रियासमभिहारे यङ् (पा॰सू॰ ३.१.२२) → वृत् यङ् → यङोऽचि च (पा॰सू॰ २.४.७४) → चकारादच्प्रत्ययं विनाऽपि यङो लुक् → वृत् → प्रत्ययलोपे प्रत्ययलक्षणम् (पा॰सू॰ १.१.६२) → सन्यङोः (पा॰सू॰ ६.१.९) → अभ्यासकार्यः → वृत् वृत् → हलादिः शेषः (पा॰सू॰ ७.४.६०) → वृ वृत् → गुणो यङ्लुकोः (पा॰सू॰ ७.४.८२) → व वृत् → उरण् रपरः (पा॰सू॰ १.१.५१) → वर् वृत् → हलादिः शेषः (पा॰सू॰ ७.४.६०) → व वृत् → चर्करीतं च (धा॰पा॰ग॰सू॰ १०८१) → अदादिगणत्वेन धातुसञ्ज्ञा → वर्तमाने लट् (पा॰सू॰ ३.२.१२३) → शेषात्कर्तरि परस्मैपदम् (पा॰सू॰ १.३.७८) → व वृत् तिप् → व वृत् ति → कर्तरि शप् (पा॰सू॰ ३.१.६८) → व वृत् शप् ति → अदिप्रभृतिभ्यः शपः (पा॰सू॰ २.४.७२) → शब्लुक् → व वृत् ति → रुग्रिकौ च लुकि (पा॰सू॰ ७.४.९१) → अभ्यासस्य पर्यायेण रुग्रिग्रीगागमाः।

(क) रुक्पक्षे: आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → व रुँक् वृत् ति → व र् वृत् ति → यङो वा (पा॰सू॰ ७.३.९४) → सार्वधातुकस्य पितः तिपो वैकल्पिक ईडागमः।

 (१) ईट्पक्षे: आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → व र् वृत् ईट् ति → व र् वृत् ई ति → वर्वृतीति।  

 (२) ईडभावे: व र् वृत् ति → पुगन्तलघूपधस्य च (पा॰सू॰ ७.३.८६) → गुणः → व र् वत् ति → उरण् रपरः (पा॰सू॰ १.१.५१) → व र् वर्त् ति → वर्वर्त्ति। झरो झरि सवर्णे (पा॰सू॰ ८.४.६५) इत्यनेन प्रथमतकारस्य वैकल्पिकलोपे वर्वर्ति इत्यपि।

(ख) रिक्पक्षे → आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → व रिक् वृत् ति → व रि वृत् ति → यङो वा (पा॰सू॰ ७.३.९४) → सार्वधातुकस्य पितः तिपो वैकल्पिक ईडागमः।

 (३) ईट्पक्षे → आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → व रि वृत् ईट् ति → व रि वृत् ई ति → वरिवृतीति।

 (४) ईडभावे → व रि वृत् ति → पुगन्तलघूपधस्य च (पा॰सू॰ ७.३.८६) → गुणः → व रि वत् ति → उरण् रपरः (पा॰सू॰ १.१.५१) → व रि वर्त् ति → वरिवर्त्ति। झरो झरि सवर्णे (पा॰सू॰ ८.४.६५) इत्यनेन प्रथमतकारस्य वैकल्पिकलोपे वरिवर्ति इत्यपि।

(ग) रीक्पक्षे → आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → व रीक् वृत् ति → व री वृत् ति → यङो वा (पा॰सू॰ ७.३.९४) → सार्वधातुकस्य पितः तिपो वैकल्पिक ईडागमः।

 (५) ईट्पक्षे → आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → व री वृत् ईट् ति → व री वृत् ई ति → वरीवृतीति।

 (६) ईडभावे → व री वृत् ति → पुगन्तलघूपधस्य च (पा॰सू॰ ७.३.८६) → गुणः → व री वत् ति → उरण् रपरः (पा॰सू॰ १.१.५१) → व री वर्त् ति → वरीवर्त्ति। झरो झरि सवर्णे (पा॰सू॰ ८.४.६५) इत्यनेन प्रथमतकारस्य वैकल्पिकलोपे वरीवर्ति इत्यपि।
