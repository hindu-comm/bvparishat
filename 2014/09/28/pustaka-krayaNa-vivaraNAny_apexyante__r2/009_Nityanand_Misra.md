“मा क्रियताम्” इत्यसाधु। “मा कारि” इत्यनेन भवितव्यम्।

“माङि लुङ्” इत्यनेन माङि सर्वलकारापवादेन लुङ् विहितः। कर्मणि प्रथमपुरुषैकवचने

मा कृ लुङ् → लुङ्लङ्लृङ्क्ष्वडुदात्तः → अडागमः → न माङ्योगे → अडागमाभावः → मा कृ लुङ् → मा कृ त → च्लि लुङि → मा कृ च्लि त → चिण् भवकर्मणोः → मा कृ चिण् त → मा कृ इ त → अचो ञ्णिति → उरण् रपरः → मा कार् इ त → चिणो लुक् → मा कार् इ → मा कारि।

यथा शिशुपालवधे (२.६३) –

इन्द्रप्रस्थगमस्तावत्कारि मा सन्तु चेदयः। आस्माकदन्तिसान्निध्याद्वामनीभूतभूरुहः॥
