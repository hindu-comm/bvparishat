Here are the derivations of ददर्श and ददृशे.



दृशिँर् प्रेक्षणे (धा॰पा॰ ९८८) → अनुबन्धलोपः → दृश् → शेषात्कर्तरि परस्मैपदम् (पा॰सू॰ १.३.७८) → परोक्षे लिट् (पा॰सू॰ ३.२.११५) → दृश् लिट् → दृश् तिप् → अनुबन्धलोपः → दृश् ति → परस्मैपदानां णलतुसुस्थलथुसणल्वमाः (पा॰सू॰ ३.४.८२) → दृश् ण → अनुबन्धलोपः → दृश् अ → लिटि धातोरनभ्यासस्य (पा॰सू॰ ६.१.८) → दृश् दृश् अ → उरत्‌ (पा॰सू॰ ७.४.६६) → उरण् रपरः (पा॰सू॰ १.१.५१) → दर्श् दृश् अ → हलादिः शेषः (पा॰सू॰ ७.४.६०) → द दृश् अ → पुगन्तलघूपधस्य च (पा॰सू॰ ७.३.८६) → उरण् रपरः (पा॰सू॰ १.१.५१) → रपरो लघूपधगुणः → द दर्श् अ → ददर्श।



दृशिँर् प्रेक्षणे (धा॰पा॰ ९८८) → अनुबन्धलोपः → दृश् → कर्तरि कर्मव्यतिहारे (पा॰सू॰ १.३.१४) → परोक्षे लिट् (पा॰सू॰ ३.२.११५) → दृश् लिट् → दृश् त → लिटस्तझयोरेशिरेच् (पा॰सू॰ ३.४.८१) → दृश् ए → लिटि धातोरनभ्यासस्य (पा॰सू॰ ६.१.८) → दृश् दृश् ए → उरत्‌ (पा॰सू॰ ७.४.६६) → उरण् रपरः (पा॰सू॰ १.१.५१) → दर्श् दृश् ए → हलादिः शेषः (पा॰सू॰ ७.४.६०) → द दृश् ए → पुगन्तलघूपधस्य च (पा॰सू॰ ७.३.८६) → लघूपधगुणप्राप्तिः → असंयोगाल्लिट् कित् (पा॰सू॰ १.२.५) → कित्त्वम् → ग्क्ङिति च (पा॰सू॰ १.१.५) → लघूपधगुणनिषेधः → द दृश् ए → ददृशे।



Question: When असंयोगाल्लिट् कित् (पा॰सू॰ १.२.५) and ग्क्ङिति च (पा॰सू॰ १.१.५) block the लघूपधगुण in ददृशे, why do they not block the same in ददर्श, for here also we have लिट् after a root not ending in a conjunct.

Answer: असंयोगाल्लिट् कित् (पा॰सू॰ १.२.५) has the अनुवृत्ति of अपित् from सार्वधातुकमपित् (पा॰सू॰ १.२.४). As a result the लिट् after a root not ending in a conjunct gets कित्त्वम् only if the substitution of लिट् is अपित्. As तिप् is not अपित् but पित्, there is no कित्त्वम् and no blocking of लघूपधगुण. Hence we have लघूपधगुण in the singular ददर्श, ददर्शिथ and ददर्श; but no लघूपधगुण in the dual and plural ददृशतुः ददृशुः ददृशथुः ददृश ददृशिव ददृशिम.
