व्याकरणशास्त्रे व्यस्थिताविभाषा इति सिद्धान्तो वर्तते।

1. व्यस्थितविभाषा is a paribhāṣā and is a sub-category of vibhāṣā.

2. The following is from paribhāṣēndu śēkhara

**vyavasthitavibhāṣayāpi kāryāṇi kriyante // Paribh_107 //**  
  
lakṣyānusārād vyavasthā bodhyā/"śācchoḥ"iti sūtre"laṭaḥ śatṛ"ityādi sūtreṣu ca bhāṣye spaṣṭā//



Please note that there is no single kind of vyavasthitavibhāṣā. vyavasthā varies from lakṣyā to lakṣyā.
