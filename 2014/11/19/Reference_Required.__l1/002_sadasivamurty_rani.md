There are two versions of this sloka. The Exact version agreeing with the enquiry is not readily available with me. But I can surely give that reference also very soon. Now I would like to place on record one available reference with slight text variation as in the following manner.

ब्रह्म सत्यं जगन्मिथ्या जीवो ब्रह्मैव नापर:।

अनेन वेद्यं सच्छास्त्रमिति वेदान्तडिण्डिम:॥

       - ब्रह्मज्ञानावलीमाला २० तमश्लोक: P.31

This is as found in the श्रीमच्छङ्कराचार्यविरचितप्रकरणग्रन्थसङ्ग्रह: of The Stotra Works of Sri Sankaracharya published Oriental Book Agency, Pune in 1939.

Soon I shall try to give the other reference also. I hope this may be of some help.
