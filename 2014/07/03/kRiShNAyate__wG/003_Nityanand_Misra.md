साधूक्तम्। कृष्ण इवाऽचरति कृष्णायते। कृष्ण सुँ → कर्तुः क्यङ् सलोपश्च (पा॰सू॰ ३.१.११) → कृष्ण सुँ क्यङ् → लशक्वतद्धिते (पा॰सू॰ १.३.८), हलन्त्यम् (पा॰सू॰ १.३.३), तस्य लोपः (पा॰सू॰ १.३.९) → कृष्ण सुँ य → सनाद्यन्ता धातवः (पा॰सू॰ ३.१.३२) → धातुसञ्ज्ञा → सुपो धातुप्रातिपदिकयोः (पा॰सू॰ २.४.७१) → सुब्लुक् → कृष्ण य → अकृत्सार्वधातुकयोर्दीर्घः (पा॰सू॰ ७.४.२५) → कृष्णा य → कृष्णाय → अनुदात्तङित आत्मनेपदम् (पा॰सू॰ १.३.१२) → आत्मनेपदप्राप्तिः → वर्तमाने लट् (पा॰सू॰ ३.२.१२३) → कृष्णाय लँट् → उपदेशेऽजनुनासिक इत् (पा॰सू॰ १.३.२), हलन्त्यम् (पा॰सू॰ १.३.३) तस्य लोपः (पा॰सू॰ १.३.९) → कृष्णाय ल् → तिप्तस्झिसिप्थस्थमिब्वस्मस्तातांझथासाथांध्वमिड्वहिमहिङ् (पा॰सू॰ ३.४.७८) → कृष्णाय त → कर्तरि शप्‌ (पा॰सू॰ ३.१.६८) → कृष्णाय शप् त → लशक्वतद्धिते (पा॰सू॰ १.३.८), हलन्त्यम् (पा॰सू॰ १.३.३), तस्य लोपः (पा॰सू॰ १.३.९) → कृष्णाय अ त → अतो गुणे (पा॰सू॰ ६.१.९७) → पररूपैकादेशः → कृष्णाय त → टित आत्मनेपदानां टेरे (पा॰सू॰ ३.४.७९) → कृष्णाय ते → कृष्णायते।
