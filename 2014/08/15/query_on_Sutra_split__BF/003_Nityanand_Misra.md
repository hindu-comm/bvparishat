Lets look at the प्रक्रिया to see the significance.  

डुदाञ् दाने (धा॰पा॰\~१०९१) → आदिर्ञिटुडवः (पा॰सू॰\~१.३.५), हलन्त्यम् (पा॰सू॰\~१.३.३), तस्य लोपः (पा॰सू॰\~१.३.९) → दा → दाधा घ्वदाप् (पा॰सू॰\~१.१.२०) → घुसञ्ज्ञा → शेषात्कर्तरि परस्मैपदम् (पा॰सू॰\~१.३.७८) → लोँट् च (पा॰सू॰\~३.३.१६२) → दा+लोँट् → उपदेशेऽजनुनासिक इत् (पा॰सू॰\~१.३.२), हलन्त्यम् (पा॰सू॰\~१.३.३), तस्य लोपः (पा॰सू॰\~१.३.९) → दा+ल् → तिप्तस्झिसिप्थस्थमिब्वस्मस्ताताञ्झथासाथान्ध्वमिड्वहिमहिङ् (पा॰सू॰\~३.४.७८) → दा+सिप् → हलन्त्यम् (पा॰सू॰\~१.३.३), तस्य लोपः (पा॰सू॰\~१.३.९) → दा+सि → सेर्ह्यपिच्च (पा॰सू॰\~३.४.८७) → दा+हि → कर्तरि शप्‌ (पा॰सू॰\~३.१.६८) → दा+शप्+हि → जुहोत्यादिभ्यः श्लुः (पा॰सू॰\~२.४.७५) → दा+हि → श्लौ (पा॰सू॰\~६.१.१०) → दा+दा+हि → पूर्वोऽभ्यासः (पा॰सू॰\~६.१.४) → ह्रस्वः (पा॰सू॰\~७.४.५९) → द+दा+हि → घ्वसोरेद्धावभ्यासलोपश्च (पा॰सू॰\~६.४.११९).

Now by the परिभाषा अलोऽन्त्यस्य (पा॰सू॰\~१.१.५२), the लोपादेश (elision) should apply to the last syllable of the अभ्यास (द), which would result in the अनिष्ट form

द्+Ø+दे+हि = X द्देहि

Question: Is there proof that लोपादेश replaces the last syllable only?  

Answer: Yes. Let’s take an example involving a घु only. The Sutra is घोर्लोपो लेटि वा (पा॰सू॰\~७.३.७०). The derivation before applying this Sutra is as follows –

डुदाञ् दाने (धा॰पा॰\~१०९१) → आदिर्ञिटुडवः (पा॰सू॰\~१.३.५), हलन्त्यम् (पा॰सू॰\~१.३.३), तस्य लोपः (पा॰सू॰\~१.३.९) → दा → दाधा घ्वदाप् (पा॰सू॰\~१.१.२०) → घुसञ्ज्ञा → शेषात्कर्तरि परस्मैपदम् (पा॰सू॰\~१.३.७८) → लिङर्थे लेट् (पा॰सू॰\~३.४.७) → दा+लेँट् → उपदेशेऽजनुनासिक इत् (पा॰सू॰\~१.३.२), हलन्त्यम् (पा॰सू॰\~१.३.३), तस्य लोपः (पा॰सू॰\~१.३.९) → दा+ल् → तिप्तस्झिसिप्थस्थमिब्वस्मस्ताताञ्झथासाथान्ध्वमिड्वहिमहिङ् (पा॰सू॰\~३.४.७८) → दा+तिप् → हलन्त्यम् (पा॰सू॰\~१.३.३), तस्य लोपः (पा॰सू॰\~१.३.९) → दा+ति → इतश्च लोपः परस्मैपदेषु (पा॰सू॰\~३.४.९७) → दा+त् → दा+त् → लेटोऽडाटौ (पा॰सू॰\~३.४.९४), आद्यन्तौ टकितौ (पा॰सू॰\~१.१.४६) → दा+अट्+त् → हलन्त्यम् (पा॰सू॰\~१.३.३), तस्य लोपः (पा॰सू॰\~१.३.९) → दा+अ+त् → कर्तरि शप्‌ (पा॰सू॰\~३.१.६८) → दा+शप्+अ+त् → जुहोत्यादिभ्यः श्लुः (पा॰सू॰\~२.४.७५) → दा+अ+त् → श्लौ (पा॰सू॰\~६.१.१०) → दा+दा+अ+त् → पूर्वोऽभ्यासः (पा॰सू॰\~६.१.४) → ह्रस्वः (पा॰सू॰\~७.४.५९) → द+दा+अ+त्. Now घोर्लोपो लेटि वा (पा॰सू॰\~७.३.७०) applies to result in optional लोप (elision) of the अङ्ग of the घु when a लेट् affix follows.

When there is no लोप, the form is द+दा+अ+त् → अकः सवर्णे दीर्घः (पा॰सू॰\~६.१.१०१) → द+दा+त् → ददात्.

When there is लोप, the form is द+दा+अ+त् → घोर्लोपो लेटि वा (पा॰सू॰\~७.३.७०), अलोऽन्त्यस्य (पा॰सू॰\~१.१.५२) → द+द्+अ+त् → ददत्. Note: When there is आट् आगम instead by लेटोऽडाटौ (पा॰सू॰\~३.४.९४), then the form is ददात् irrespective of the optional लोप by घोर्लोपो लेटि वा (पा॰सू॰\~७.३.७०).

So coming back to the original question. The इष्ट form as we all know is देहि. As in रूपं देहि जयं देहि यशो देहि द्विषो जहि (this can be heard in the reverberating voice of स्वामी मुक्तानन्द of गणेशपुरी).

So how come the entire अभ्यास gets elided by घ्वसोरेद्धावभ्यासलोपश्च (पा॰सू॰\~६.४.११९) in the case of द+दा+हि?

The Kashika says that this लोपादेश is शित्. Then by अनेकाल्शित्सर्वस्य (पा॰सू॰\~१.१.५५), an अनेकाल् शित् आदेश replaces what is to be replaced entirely (सर्वस्य) and not just its last syllable. But one doubt still remains. How come लोप is अनेकाल्? लोप is actually शून्याल्. But शून्य is also अनेक (not one). अनेक in Sanskrit means not only many, but anything that is not one (including zero also). So as this लोपादेश is शित्, and since all लोपादेशs are अनेकाल्, the परिभाषा अनेकाल्शित्सर्वस्य (पा॰सू॰\~१.१.५५) blocks the earlier परिभाषा अलोऽन्त्यस्य (पा॰सू॰\~१.१.५२) and the entire अभ्यास gets elided.

Hence the इष्ट form is

द+दा+हि → घ्वसोरेद्धावभ्यासलोपश्च (पा॰सू॰\~६.४.११९) → Ø+दे+हि → देहि.

Explaining, Kashika, the न्यास says that the Sutra घ्वसोरेद्धावभ्यासलोपश्च has two शकारs.

घ्वसोरेद्धावभ्यासलोपश्च = घ्वसोः एत् हौ अभ्यासलोपः श् च

The second श् is an adjective of अभ्यासलोपः and means that this अभ्यासलोप is a शित्, that is what the न्यासकार means.
