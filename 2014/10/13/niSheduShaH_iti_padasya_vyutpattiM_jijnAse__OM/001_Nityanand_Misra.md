नि षद्ऌँ विशरणगत्यवसादनेषु (धा॰पा॰ ८५४) → अनुबन्धलोपः → नि षद् → धात्वादेः षः सः (पा॰सू॰ ६.१.६४) → नि सद् → परोक्षे लिट् (पा॰सू॰ ३.२.११५) → नि सद् लिट् → भाषायां सदवसश्रुवः (पा॰सू॰ ३.२.१०८) → नि सद् क्वसुँ → अनुबन्धलोपः → नि सद् वस् → वस्वेकाजाद्घसाम् (पा॰सू॰ ७.२.६७) → इडागमः → नि सद् इट् वस् → अनुबन्धलोपः → नि सद् इ वस् → लिटि धातोरनभ्यासस्य (पा॰सू॰ ६.१.८) → नि सद् सद् इ वस् → हलादिः शेषः (पा॰सू॰ ७.४.६०) → नि स सद् इ वस् → अत एकहल्मध्येऽनादेशादेर्लिटि (पा॰सू॰ ६.४.१२०) → अभ्यासलोप एत्वञ्च → नि सेद् इ वस् → आदेशप्रत्यययोः (पा॰सू॰ ८.३.५९) → षत्वम् → नि षेद् इ वस् → निषेदिवस्।

‘निषेदिवस्’ इति प्रातिपदिकम्। तस्य पुल्लिँङ्गे नपुंसकलिङ्गे च पञ्चम्येकवचने षष्ठ्येकवचने च ‘निषेदुषः’ इति रूपम्।

निषेदिवस् ङसिँ/ङस् → अनुबन्धलोपः → निषेदिवस् अस् → यचि भम् (पा॰सू॰ १.४.१८) → भसञ्ज्ञा → वसोः सम्प्रसारणम् (पा॰सू॰ ६.४.१३१) → निषेदि उ अस् अस् → अकृतव्यूहाः पाणिनीयाः (प॰शे॰ ५६) → सम्प्रसारण इडागमनिवर्तनम् → निषेद् उ अस् अस् → सम्प्रसारणाच्च (पा॰सू॰ ६.१.१०८) → पूर्वरूपैकादेशः → निषेद् उस् अस् → आदेशप्रत्यययोः (पा॰सू॰ ८.३.५९) → षत्वम् → निषेद् उष् अस् → निषेदुषस् → ससजुषो रुः (पा॰सू॰ ८.२.६६) → निषेदुषरुँ → खरवसानयोर्विसर्जनीयः (पा॰सू॰ ८.३.१५) → निषेदुषः।

परिभाषेन्दुशेखरे नागेशभट्टपादाः –

नन्वेवं ‘सेदुषः’ इत्यादौ क्वसोन्तरङ्गत्वादिटि। ततः सम्प्रसारणेऽपीटः श्रवणापत्तिरिति चेत्। अत्र केचित् –

अकृतव्यूहाः पाणिनीयाः॥ ५६ ॥

यद्येवं न स्यात् तर्हि ‘xसेद्युषः’/‘xनिषेद्युषः’ इत्यनिष्टरूपौ सयाताम्।
