The above statement is incorrect. प्राप्य is not a wrong usage even if the sense is णिजन्त. प्रापय्य and प्राप्य as both valid forms for प्र+आप्+णिच्+ल्यप्.

Refer विभाषाऽऽपः (पा॰सू॰ ६.४.५७) and commentaries on the same.

णेरयादेशे प्रापय्य इति रूपम्।

प्र आप् → हेतुमति च (पा॰सू॰ ३.१.२६) → प्र आप् णिच् → अनुबन्धलोपः → प्र आप् इ → प्र आपि → सनाद्यन्ता धातवः (पा॰सू॰ ३.१.३२) → धातुसञ्ज्ञा → समानकर्तृकयोः पूर्वकाले (पा॰सू॰ ३.४.२१) → प्र आपि क्त्वा → समासेऽनञ्पूर्वे क्त्वो ल्यप्‌ (पा॰सू॰ ७.१.३७) → प्र आपि ल्यप् → अनुबन्धलोपः → प्र आपि य → विभाषाऽऽपः (पा॰सू॰ ६.४.५७) → णेर्वैकल्पिकोऽयादेशः → प्र आपय् य → प्रापय्य।

अयादेशाभावे पक्षे प्राप्य इति रूपम् –

प्र आपि य → णेरनिटि (पा॰सू॰ ६.४.५१) → प्र आप् य → प्राप्य।

In addition, णिजन्त usage does not appear to be the case either (see below).

The verse has जडनिधि in the accusative case (जडनिधिम्). The above explanation has it in the nominative case. It should be taken as त्वं प्रत्यग्ब्रह्मामृतरसनिधिं विश्राम्यसि इयं \[जाह्नवी तु\] जडनिधिं \[प्राप्य विश्राम्यति\]. Therefore प्राप्य = on reaching. Hence no scope for णिजन्त in प्राप्य above. Even if explained as णिजन्त, both प्राप्य/प्रापय्य are correct forms as explained.
