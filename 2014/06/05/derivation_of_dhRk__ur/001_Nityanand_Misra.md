Please excuse any typos/errors.  
  

पिनाकेन धृष्णोतीति पिनाकधृक्। “ऋत्विग्दधृक्स्रग्दिगुष्णिगञ्चुयुजिक्रुञ्चां च” (पा॰सू॰ ३.२.५९) इति सूत्रे निपातदर्शनात् “धृष्” धातोः (ञिधृषाँ प्रागल्भ्ये, धा॰पा॰ १२६९) सुबन्तोपपदे क्विन् प्रत्ययो भवति। प्रकृतसूत्रे “सुपि” इति “सुपि स्थः” (पा॰सू॰ ३.२.४) इत्यतोऽनुवर्त्यते। पिनाक टा धृष् क्विँन् → लशक्वतद्धिते (पा॰सू॰ १.३.८), उपदेशेऽजनुनासिक इत् (पा॰सू॰ १.३.२), हलन्त्यम् (पा॰सू॰ १.३.३) → ककारेकारनकाराणामित्संज्ञा → तस्य लोपः (पा॰सू॰ १.३.९) → पिनाक टा धृष् व् → वेरपृक्तस्य (पा॰सू॰ ६.१.६७) → वलोपः → पिनाक टा धृष् → उपपदमतिङ् (पा॰सू॰ २.२.१९) → समासः → कृत्तद्धितसमासाश्च (पा॰सू॰ १.२.४६) → प्रातिपदिकसंज्ञा → सुपो धातुप्रातिपदिकयोः (पा॰सू॰ २.४.७१) → उपपदविभक्तिर्लोपः → पिनाक धृष् → पिनाकधृष् → विभक्तिकार्यम् →
स्वौजसमौट्छष्टाभ्याम्भिस्ङेभ्याम्भ्यस्ङसिभ्याम्भ्यस्ङसोसाम्ङ्योस्सुप्‌ (पा॰सू॰ ४.१.२) → पिनाकधृष् सुँ → हल्ङ्याब्भ्यो दीर्घात्सुतिस्यपृक्तं हल् (पा॰सू॰ ६.१.६८) → सुलोपः → पिनाकधृष् → झलां जशोऽन्ते (पा॰सू॰ ८.२.३९) → पिनाकधृड् → क्विन्प्रत्ययस्य कुः (पा॰सू॰ ८.२.६२) → कुत्वम् → पिनाकधृग् → वाऽवसाने (पा॰सू॰ ८.४.५६) → वैकल्पिकचर्त्वम् → पिनाकधृक्, पिनाकधृग्।

यद्वा “ऋत्विग्दधृक्स्रग्दिगुष्णिगञ्चुयुजिक्रुञ्चां च” (पा॰सू॰ ३.२.५९) इति सूत्रे चकारग्रहणात्क्वचिदन्यत्रापि क्विन् प्रत्ययो भवति। तेन पिनाकेन धर्षयतीति धर्षतीति वा पिनाकधृक्। “धृष्” धातोः (धृषँ प्रसहने, धा॰पा॰ १८५०) सुबन्तोपपदे क्विन्। “आधृषाद्वा” (धा॰पा॰ग॰सू॰ १८०६) इत्यनेन णिज्वैकल्पिकः। णिजभावपक्षे “पिनाक टा धृष् क्विँन्” इति स्थिते प्रक्रिया पूर्ववत्। णिच्पक्षे च “पिनाक टा धृष् णिच् क्विँन्” इति स्थिते “णेरनिटि” (पा॰सू॰ ६.४.५१) इत्यनेन णिलोपे “पिनाक टा धृष् क्विँन्” इति जाते प्रक्रिया पूर्ववदेव।

अन्यत्र च “–धृक्” रूपाणि दृश्यन्ते। यथा विष्णुसहस्रनामस्तोत्रे “विश्वधृक्” इति नाम पठितम् (“सुप्रसादः प्रसन्नात्मा विश्वधृग्विश्वभुग्विभुः”)। विश्वं धृष्णोतीति विश्वधृक्। एवमेव “शूलेन धृष्णोतीति धर्षयतीति धर्षतीति वा शूलधृक्”। सुश्रुतसंहितायाम् “वायुर्यो वक्त्रसञ्चारी स पाणो नाम देहधृक्” इति पठितम्। तत्र देहे धर्जतीति देहधृक्। “धृज्” धातोः (धृजँ गतौ, धा॰पा॰ १८५०) सुबन्तोपपदे पूर्ववत्क्विन्प्रत्यये सर्वापहारिलोपे “देहधृज्” इति जाते विभक्तिकार्ये “चोः कुः” (पा॰सू॰ ८.२.३०) इत्यनेन कुत्वे “देहधृग्” इति जाते “वाऽवसाने” (पा॰सू॰ ८.४.५६) इत्यनेन वैकल्पिकचर्त्वे “देहधृक्/देहधृग्” इति रूपद्वयम्।
