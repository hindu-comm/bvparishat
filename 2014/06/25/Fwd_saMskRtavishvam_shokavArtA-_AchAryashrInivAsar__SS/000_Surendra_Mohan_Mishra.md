//aacaarya-s'rinivaasarathamahodayaanaam mahaaprayaane //  
  
Om  
  
yasáh-kaaye 'dya ye s'lokyaah kaaye s'laaghyaa ihaa 'vasan /  
avantikaayaam kaavyaani yeshaam keshaam na sampadah //  
utkalebhyah samaayaataa ujjigiishava eva vaa /  
ujjayinyaam mahaakaalam kavi-pandita-sattamaah ! //  
aho kaalajayii kanthah aho bhaavaa aho kalaa /  
aho prasanna-gaambhiiryam sarvatraa 'nanvayah kila //  
kaalidaasa-samaaroha ujjigaasati yadyasáh /  
aho sáantarase sarvarasaa ekakalevare ! //  
s'riyaam kavitvasya nivaasa aasate s'riyaam tu sáastrasya nivaasa aasate /  
rathe sáriire jayatiiha sékharah s'riisrinivaaso 'janitaarir aksharah ! //  
yasya vetti jagannaatho mahaakaalo 'pi vaa gatim /  
ke vayam tasya vijnaatum nirvritim nirjare 'dhvani ! //  
  
gunamugdhah kas'çit  
  
surendramohanamis'rah  
kurukshetram
