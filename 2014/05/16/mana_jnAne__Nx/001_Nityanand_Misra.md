On Friday, May 16, 2014 11:05:39 AM UTC+8, Swami Suprajnananda wrote:

> 
> > मन ज्ञाने धातो: मध्यमपुरुषलोडन्तरूपं किं भवति? मन्यथा इति वा? >
> 

  

मन्यस्व इति पाणिनीयरूपम्। यथा रामायणे भट्टिकाव्ये च –

तत्त्रिकालहितं वाक्यं धर्म्यमर्थानुबन्धि च।

मन्यस्व नरदेवाय जानकी प्रतिदीयताम्॥ वा॰रा॰ ५.५१.२१ ॥

शिला तरिष्यत्युदके न पर्णं ध्वान्तं रवेः स्यन्त्स्यति वह्निरिन्दोः।

जेता परोऽहं युधि जेष्यमाणस्तुल्यानि मन्यस्व पुलस्त्यनप्तः॥ भ॰का॰ १२.१७ ॥

  
  


> 
> > अस्य धातो: लोट्रूपं सर्वं दीयतामिति प्रार्थ्यते । >
> 
> >   
> > 
> > 

  

ससिद्धिरूपाणि गृह्णन्तु –

मन्यताम् मन्येताम् मन्यन्ताम्

मन्यस्व मन्येथाम् मन्यध्वम्

मन्यै मन्यावहै मन्यामहै

मन् -> ३-३-१६२ लोँट् च -> मन् लोँट् -> मन् ल -> ३-४-७८ तिप्तस्झिसिप्थस्थमिब्वस्मस्ताऽतांझथासाथाम्ध्वमिड्वहिमहिङ्

(१) मन् त -> ३-१-६९ दिवादिभ्यः श्यन् -> मन् शयन् त -> मन् य त -> ३-४-७९ टित आत्मनेपदानां टेरे -> मन् य ते -> ३-४-९० आमेतः -> मन् य त आम् -> मन्यताम्

(२) मन् आताम् -> ३-१-६९ दिवादिभ्यः श्यन् -> मन् शयन् आताम् -> मन् य आताम् -> ३-४-८५ लोटो लङ्वत्‌ -> लङ्वत्त्वम् -> ७-२-८१ आतो ङितः -> मन् य इय् ताम् -> ६-१-६६ लोपो व्योर्वलि -> मन् य इ ताम् -> ६-१-८७ आद्गुणः -> मन् ये ताम् -> ३-४-७९ टित आत्मनेपदानां टेरे -> मन् ये ते -> ३-४-९० आमेतः -> मन् ये ताम् -> मन्येताम्

(३) मन् झ -> ३-१-६९ दिवादिभ्यः श्यन् -> मन् शयन् झ -> मन् य झ -> ७-१-३ झोऽन्तः -> मन् य अन्त -> ६-१-९७ अतो गुणे -> मन् यन्त -> ३-४-७९ टित आत्मनेपदानां टेरे -> मन् यन्ते -> ३-४-९० आमेतः -> मन् यन्ताम् -> मन्यन्ताम्

(४) मन् थास् -> ३-१-६९ दिवादिभ्यः श्यन् -> मन् शयन् थास् -> मन् य थास् -> ३-४-८० थासस्से -> मन् य से -> ३-४-९१ सवाभ्यां वामौ -> मन् य स्व -> मन्यस्व

(५) मन् आथाम् -> ३-१-६९ दिवादिभ्यः श्यन् -> मन् शयन् आथाम् -> मन् य आथाम् -> ३-४-८५ लोटो लङ्वत्‌ -> लङ्वत्त्वम् -> ७-२-८१ आतो ङितः -> मन् य इय् थाम् -> ६-१-६६ लोपो व्योर्वलि -> मन् य इ थाम् -> ६-१-८७ आद्गुणः -> मन् ये थाम् -> ३-४-७९ टित आत्मनेपदानां टेरे -> मन् ये थे -> ३-४-९० आमेतः -> मन् ये थाम् -> मन्येथाम्

(६) मन् ध्वम् -> ३-१-६९ दिवादिभ्यः श्यन् -> मन् शयन् ध्वम् -> मन् य ध्वम् -> ३-४-७९ टित आत्मनेपदानां टेरे -> मन् य ध्वे -> ३-४-९१ सवाभ्यां वामौ -> मन् य ध्वम् -> मन्यध्वम्

(७) मन् इट् -> मन् इ -> ३-१-६९ दिवादिभ्यः श्यन् -> मन् शयन् इ -> मन् य इ -> ३-४-७९ टित आत्मनेपदानां टेरे -> मन् य ए -> ३-४-९३ एत ऐ -> मन् य ऐ -> ६-१-८८ वृद्धिरेचि -> मन् यै -> मन्यै

(८) मन् वहिङ् -> मन् वहि -> ३-१-६९ दिवादिभ्यः श्यन् -> मन् शयन् वहि -> मन् य वहि -> ७-३-१०१ अतो दीर्घो यञि -> मन् या वहि -> ३-४-७९ टित आत्मनेपदानां टेरे -> मन् या वहे -> ३-४-९३ एत ऐ -> मन् या वहै -> मन्यावहै

(९) मन् महिङ् -> मन् महि -> ३-१-६९ दिवादिभ्यः श्यन् -> मन् शयन् महि -> मन् य महि -> ७-३-१०१ अतो दीर्घो यञि -> मन् या महि -> ३-४-७९ टित आत्मनेपदानां टेरे -> मन् या महे -> ३-४-९३ एत ऐ -> मन् या महै -> मन्यामहै
