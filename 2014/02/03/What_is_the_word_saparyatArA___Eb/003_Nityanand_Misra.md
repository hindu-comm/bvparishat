As रामनुजाचार्य महोदय has said, सपर्यत is the लोट् मध्यपुरुष बहुवचन form coming from सपर पूजायाम्

सपर पूजायाम् -\> सपर -\> 3-1-27 कण्ड्वादिभ्यो यक् -\> सपर यक् -\> 1-3-3 हलन्त्यम्, 1-3-9 तस्य लोपः -\> अनुबन्धलोपः -\> सपर य -\> 3-4-114 आर्धधातुकं शेषः -\> ‘य’ इत्यस्य आर्धधातुकसंज्ञा -\> 6-4-48 अतो लोपः -\> अलोपः -\> सपर् य -\> सपर्य -\> 3-1-32 सनाद्यन्ता धातवः -\> धातुसंज्ञा -\> 3-3-162 लोट् च -\> सपर्य लोँट् -\> 1-3-2 उपदेशेऽजनुनासिक इत्, 1-3-3 हलन्त्यम्, 1-3-9 तस्य लोपः -\> अनुबन्धलोपः -\> सपर्य ल् -\> 3-4-78 तिप्तस्झिसिप्थस्थमिब्वस्मस्तातांझथासाथांध्वमिड्वहिमहिङ् -\> 1-3-78 शेषात्कर्तरि परस्मैपदम् -\> परस्मैपदप्राप्तिः -\> सपर्य थ -\> 3-4-101 तस्थस्थमिपाम् तान्तन्तामः -\> थस्य तादेशः -\> सपर्य त -\> सपर्यत

Question: How does यक् get the आर्धधातुकसंज्ञा which is required for अलोप in सपर य -\> सपर्य? आर्धधातुकसंज्ञा is intended only for suffixes ordained after what is already defined as धातु (e.g. those suffixes read from 3-1-91 धातोः till end of chapter 3), whereas the above uses धातुसंज्ञा after the elision of अ?

Answer: Actually कण्ड्वादि already have the धातु nomenclature. This is because the सूत्र which ordains यक्, i.e. 3-1-27 कण्ड्वादिभ्यो यक्, has the अनुवृत्ति of धातोः from 3-1-22 धातोरेकाचो हलादेः क्रियासमभिहारे यङ्. With this अनुवृत्ति and विभक्तिपरिणाम, 3-1-27 reads कण्ड्वादिभ्यो धातुभ्यो \[परः\] यक् \[प्रत्ययः स्यात्\]. So कण्ड्वादि have already been given the धातुसंज्ञा by Panini. This अनुवृत्ति of धातोः from 3-1-22 runs from 3-1-23 till 3-1-28. This is how यङ्, णिच्, यक्, आय all get आर्धधातुकसंज्ञा despite coming before 3-1-91. This implied धातुसंज्ञा is not shown above explicitly since no separate सूत्र is involved and अनुवृत्तिs are not shown. In this प्रक्रिया, the धातुसंज्ञा comes twice, once from 3-1-27 कण्ड्वादिभ्यो \[धातुभ्यो\] यक् and a second time from 3-1-32 सनाद्यन्ता धातवः.
