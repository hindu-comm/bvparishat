यथा॑विधो॒यं क्रम॒स्तदु॑च्यते -

  

अत्र॒ क्रमे॒ किमपि॑ सू॒क्तं गृ॑ह्णातु। तस्य॑ प॒ठ्यमा॑न॒म् इति॑ स॒ञ्ज्ञा। तस्यो॒परि॑ लिखि॒तानां॑ सू॒क्तानां॑ पठि॒तानीति॑ स॒ञ्ज्ञा। अ॒धो लि॑खि॒ताना॒म् अप॑ठिता॒नीति॑। प॒ठि॒तेषु॑ सू॒क्तेषु॒ ये शब्दा॒स्तेषां॒ परि॑चिता॒ इति॑ स॒ञ्ज्ञा। ये न प॑ठि॒तास्तेषा॒म् अप॑रिचिता॒ इति॑। एक॑प्रातिपदिकानि रू॒पाण्ये॑क॒त्वेन॑ गण्यन्ते॒ तथै॒वैक॑धातू॒न्यपीत्य॒यम् अत्र॑ विशे॒षः। प॒ठ्यमा॑नस्याध॒स्तनं॒ यत् सू॒क्तं तद् अप॑ठिते॒ष्वल्पा॑परिचिततमम्। अप॑ठितेषु॒ तदे॒वास्ति॒ यस्मि॒न्न् अल्प॑तमः समू॒होप॑रिचितानां॒ शब्दा॑नां प॒ठ्यत॒ इत्य॑भिप्रा॒यः। यच्च॑ प्रथ॒मं लि॑खि॒तं सू॒क्तं तत् सर्वे॒ष्वल्प॑शब्दसमूहतमम्। अ॒यमे॒वास्य क्रम॑स्याधा॒रः। तत्र॑ सू॒क्तैः स॒ह तेषा॑म् ऋक्स॒ङ्ख्या न॑वशब्दस॒ङ्ख्या च॑ लिखि॒ते। य॒थे॒ष्टम् उप॑भुज्यताम्।

  

Pick a hymn in this index. Call it the current hymn. All hymns written above it are called read. All written below are called unread. All words (various verbal and noun forms counted as one) in the read hymns are called known. Others are unknown. The hymn next to the current is, among the unread, the one with the smallest set of unknown words. The first hymn written is the one, among all, with the smallest set of words.
