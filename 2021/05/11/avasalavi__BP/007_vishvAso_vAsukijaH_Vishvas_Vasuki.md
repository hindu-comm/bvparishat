नैवम् - अर्थवैपरीत्यात्। यतः प्रसलवि = प्रदक्षिणम्।

  

  

**प्रयोगा** रुचिराः -  
  
**कात्यायनश्रौतसूत्रेषु** -अपसलविसृष्टया रज्ज्वा परितत्याऽपेतो  
**द्राह्यायनगृह्ये** -उदपात्राण्यपसलवि कर्षूषु

  
**गोभिलगृह्ये** -  

``` अथैतद्बलिशेषमद्भिरभ्यासिच्यावसलवि दक्षिणा निनयेत्तत्पितृभ्यो भवति १२ सव्येनैव पाणिनोदपात्रं गृहीत्वावसलवि पूर्वस्यां कर्ष्वां दर्भेषु निनयेत्पितुर्नाम गृहीत्वासाववनेनिक्ष्व ये चात्र त्वानु याँ श्च त्वमनु तस्मै ते स्वधेति ६ सव्येनैव पाणिना दर्वीं गृहीत्वा सन्नीतात्तृतीयमात्रमवदायावसलवि पूर्वस्यां कर्ष्वां दर्भेषु निदध्यात्पितुर्नाम गृहीत्वासावेष ते पिण्डो ये चात्र त्वानु सव्येनैव पाणिना दर्भपिञ्जूलीं गृहीत्वावसलवि पूर्वस्यां कर्ष्वां पिण्डे निदध्यात्पितुर्नाम गृहीत्वासावेतत्त आञ्जनं ये चात्र त्वानु सव्येनैव पाणिना सूत्रतन्तुं गृहीत्वावसलवि पूर्वस्यां कर्ष्वां पिण्डे निदध्यात्पितुर्नाम गृहीत्वासावेतत्ते वासो ये चात्र त्वानु सव्येनैव पाणिनोदपात्रं गृहीत्वावसलवि पिण्डान्परिषिञ्चेदूर्जं वहन्तीरिति

पारस्करगृह्ये - 
मूत्रमासिच्यापसलवि त्रिः परिषिञ्चन्परीयात् ।

शाङ्खायनश्रौते - ```

``` तां संहत्य मौञ्जीभिः स्यान्द्याभिर् विवयन्ति द्विगुणाभिः प्रसलविसृष्टाभिः ९ अप्रदक्षिणं त्रिः १५ प्रसलवि तूष्णीं तत ऊर्ध्वम् १६ 
वेदिं त्रिः प्रसलवि परिवर्तमान एतमाखणं विध्यति ४ 
```

**शतपथब्राह्मणे** - तद्विधायापसलविसृष्टाभि स्पन्द्याभिः पर्यातनोत्यपसलवि पित्र्यं हि कर्म तदपसलवि पर्याहृत्योत्तरतः प्रतीचीम् प्रथमां सीताम् कृषति वायुः पुनात्विति अथापसलवि त्रिर्धून्वन्ति पुनः प्रसलवि त्रिर्धून्वन्ति

``` अपसलवि त्रिः परिस्तृणन्पर्येति सो ऽपसलवि त्रिः परिस्तीर्य 
यावत्प्रस्तरभाजनं तावत्परिशिनष्ट्यथ पुनः प्रसलवि त्रिः पर्येति 
यत् पुनः प्रसलवि त्रिः पर्येति तद् यान् एवामूंस् त्रयान् पितॄन् अन्ववागात् - 
तेभ्य एवैतत् पुनर् अपोदेतीमं स्वं लोकमभि - तस्मात्पुनः प्रसलवि त्रिः पर्येति 
```

``` यद्यध्वर्युर्वा यजमानो वा स उदपात्रमादायापसलवि त्रिः परिषिञ्चन् पर्येति 
स यजमानस्य पितरम् अवनेजयत्य् ```

``` अथोदपात्रमादाय । पुनः प्रसलवि त्रिः परिषिञ्चन्पर्येति स यजमानस्य पितरम् अवनेजयत्य् …तद् यत् पुनः प्रसलवि त्रिः परिषिञ्चन् पर्येति - प्रसलवि न इदं कर्मानुसन्तिष्ठाता इति - तस्मात् पुनः प्रसलवि त्रिः परिषिञ्चन् पर्येति

अथापसलवि त्रिः परियन्ति । 

अथ पुनः प्रसलवि त्रिः परियन्ति   
दक्षिणान् ऊरून् उपाघ्नाना एतेनैव मन्त्रेण।  
तद् यत् पुनः प्रसलवि त्रिः परियन्ति - प्रसलवि न इदं कर्मानुसन्तिष्ठाता इति -  
तस्मात् पुनः प्रसलवि त्रिः परियन्ति   
```

``` ```
