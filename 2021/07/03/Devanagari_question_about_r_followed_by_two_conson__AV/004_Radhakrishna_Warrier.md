Interesting question. I believe the answer is that if the repha symbol is placed above any letter, including a conjunct letter, the repha should be pronounced first. If we writeअर्घ्य the 'r' should be pronounced before the conjunct letter ghya. On the other hand if we want to write ayrgha in Devanagari, we will have to write it asअय्र्घ or asअय् र्घ .
