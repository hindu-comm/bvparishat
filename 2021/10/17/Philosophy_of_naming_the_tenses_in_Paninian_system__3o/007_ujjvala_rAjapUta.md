> but i couldn't get the author's logic in saying that ṅ-lakāras > wouldn't allow one to apply **टित आत्मनेपदानां टेरे** (३.४.७९) when in > fact this sutra is not used for loṭ...  

  

क आ॑ह महोदय॒ न सूत्र॑म् इ॒दल्ँ लो॒टि प्रव॑र्तत॒ इति॑? इ॒माम् प॑श्यतु-।

  

*तन् + लोट् \[लोट् च [३.३.१६२](https://ashtadhyayi.com/sutraani/3/3/162) इति लोट्-लकारः\]  
→ तन् + वहि \[तिप्तस्..[३.४.७८](https://ashtadhyayi.com/sutraani/3/4/78) इति वहि-प्रत्ययः\]  
→ तन् + उ + वहि \[तनादिकृञ्भ्यः उः [३.१.७९](https://ashtadhyayi.com/sutraani/3/1/79) इति उ-प्रत्ययः\]  
→ तन् + उ + वहे \[टित आत्मनेपदानां टेरे [३.४.७९](https://ashtadhyayi.com/sutraani/3/4/79) इति इकारस्य एकारादेशः\]  
→ तन् + उ + वहै \[एत ऐ [३.४.९३](https://ashtadhyayi.com/sutraani/3/4/93) इति एकारस्य ऐकारादेशः\]  
→ तन् + उ + आट् + वहै \[आडुत्तमस्य पिच्च [३.४.९२](https://ashtadhyayi.com/sutraani/3/4/92) इति आडागमः\]  
→ तन् + ओ + आ + वहै \[आट्-आगमस्य पित्-त्वात् सार्वधातुकार्धधातुकयोः [७.३.८४](https://ashtadhyayi.com/sutraani/7/3/84) इति गुणः\]  
→ तन् + अव् + आ + वहै \[एचोऽयवावावः [६.१.७८](https://ashtadhyayi.com/sutraani/6/1/78) इति अवादेशः\]  
*

*→ तनवावहै*
