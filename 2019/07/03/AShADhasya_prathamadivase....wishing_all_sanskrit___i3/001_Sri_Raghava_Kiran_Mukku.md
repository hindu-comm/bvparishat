# आषाढस्य प्रथमदिवसे... 

… मुम्बपुर्यां निवासात्

द्रक्ष्ये मेघं जलनिधिनिभं सर्वसिञ्चन्तमद्य।

साकं वर्षा जलगतगजप्रेक्षणीयैर्महाभ्रैः

संप्राप्तेयं जनकसदनं संगतेव प्रसूत्यै॥
