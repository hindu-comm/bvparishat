\>**जातकचन्द्रिका**- recommends ज्ञान ( ज्ञानी / ज्ञानवान् / ज्ञानेश्वरः -- ज्ञानमयी / ज्ञानेश्वरी / ज्ञानवती ) for अर्द्रातृतीयचरणम् ( ङ ) --

  

Dear Sir,

  Namaste.

  The following information I preserved in Feb. 2014 fromदाते पञ्चाङ्ग published from Maharashtra.

  

Since no name can begin with**ङ**or**ञा**, the Maharashtrian Panchanga Makers have decided that these letters be replaced respectively withगं ज्ञा.  

  

Moreover, what about**षा णा -**हस्त नक्षत्र (कन्या राशि) ? दाते पञ्चाङ्ग still preserves these.

In North India, I have not heard any name starting withष, although in South India we do have examples.

Best regards

Narayan Prasad
