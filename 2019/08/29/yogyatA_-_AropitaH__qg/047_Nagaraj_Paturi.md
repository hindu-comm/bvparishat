\> Only that due to his karma certain indriya/s do not function or function below par.

  

When there is no function, there is no question of yogyataa. in such cases Yogtataa is hypothetical . If the indriya would have been functional it would have detected /sensed only this vishaya.

  

That 'only' is what is being talked as intrinsic, anaadi etc.
