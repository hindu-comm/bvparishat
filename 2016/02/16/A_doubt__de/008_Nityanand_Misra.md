The fact is that तान्तम् is the best word to explain अतमि as both are from the same root. 

  

तमुँ (धा॰पा॰ १२०२) → तम् → निष्ठा (पा॰सू॰ ३.२.१०२), तयोरेव कृत्यक्तखलर्थाः (पा॰सू॰ ३.४.७०) → तम् क्त → तम् त → अनुनासिकस्य क्विझलोः क्ङिति (पा॰सू॰ ६.४.१५) → उपधादीर्घत्वम् → ताम् त → मोऽनुस्वारः (पा॰सू॰ ८.३.२३) → तां त → अनुस्वारस्य ययि परसवर्णः (पा॰सू॰ ८.४.५८) → तान् त → तान्त → विभक्तिकार्यम् → तान्तम्।



तमुँ (धा॰पा॰ १२०२) → तम् → भावकर्मणोः (पा॰सू॰ १.३.१३) → लुङ् (पा॰सू॰ ३.२.११०) → तम् लुङ् → तिप्तस्झिसिप्थस्थमिब्वस्मस्ताताञ्झथासाथान्ध्वमिड्वहिमहिङ् (पा॰सू॰ ३.४.७८) → तम् त → लुङ्लङ्लृङ्क्ष्वडुदात्तः (पा॰सू॰ ६.४.७१) → आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → अट् तम् त → अ तम् त → च्लि लुङि (पा॰सू॰ ३.१.४३) → अ तम् च्लि त → चिण् भावकर्मणोः (पा॰सू॰ ३.१.६६) → अ तम् चिण् त → अ तम् इ त → अत उपधायाः (पा॰सू॰ ७.२.११६) → उपधावृद्धिप्राप्तिः → नोदात्तोपदेशस्य मान्तस्यानाचमेः (पा॰सू॰ ७.३.३४) → उपधावृद्धिनिषेधः → चिणो लुक् (पा॰सू॰ ६.४.१०४) → तस्य लुक् → अ तम् इ → अतमि।

  

However, something is wrong in the explanationतमेर्ण्यन्ताद्भावे लुङ्byMallinātha (as printed in the NSP Seventh edition, 1917, p. 185). If there isṇyanta, theṇyanta root will besakarmaka (make somebody to tire, or make somebody wish). For a sakarmaka root,bhava-prayoga is not possible byलः कर्मणि च भावे चाकर्मकेभ्यः (पा॰सू॰ ३.४.६९), onlykartṛ-prayoga andkarma-prayoga are possible. If taken asṇyantakarma-prayoga, the form would not beनितम्बिनीभिः अतमि but ratherनितम्बिन्यः अतमिषत (‘the broad-hipped \[women\] were made to be tired’). So it is notतमेर्ण्यन्ताद्भावे लुङ् but rather तमेः प्रकृत्यां भावे लुङ्. As the original root तम् is akarmaka,bhava-prayogais possible from it. The ण्यन्तात् part could be a scribal/editorial error. Other editions of the commentary (Vavilla Ramaswamy?) may be consulted.
