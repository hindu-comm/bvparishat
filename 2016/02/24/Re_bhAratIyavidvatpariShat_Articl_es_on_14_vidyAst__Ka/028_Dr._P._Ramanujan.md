फलितज्यौतिषस्य गणितज्यौतिषस्य च अन्तरं विद्यते ।

वेदाङ्गेषु यज्ञानां कालविधानशास्त्रत्वात् गणितज्यौतिषम् उपयुज्यते ।

तैत्तिरीये (3.3.8)

|                                                                                                             |     |     |     |     | |-------------------------------------------------------------------------------------------------------------|-----|-----|-----|-----| | अह्नां वि॒धान्या॑मेकाष्ट॒काया॑मपू॒पं चतुः॑शरावम्प॒क्त्वा प्रा॒तरे॒तेन॒ कख्ष॒मुपौ॑षे॒द्यदि॑                 |    |    |    |    | | दह॑ति पुण्य॒सम॑म्भवति॒ यदि॒ न दह॑ति पाप॒सम॑मे॒तेन॑ ह स्म॒ वा ऋ॑षयः पु॒रा वि॒ज्ञाने॑न दीर्घस॒त्त्रमुप॑ यन्ति |     |     |     |     |

इति श्रूयते ।

नवग्रहाणां पञ्चाङ्गेषु मध्ये वार, योग, करणानां च विषयः वेदेषु न दृश्यते । जातकस्य च तथा ।

विमृश्य विद्वांसः प्रमाणम् ॥



रामानुजः
