मान्याः। एवमस्ति ग्रन्थद्वये दण्डस्य नियमाः।  

  
Vasiṣṭhadharmasṃṛti reads:  
  
Va.11.49 [garbha.a.s.tame.su](http://garbha.a.s.tame.su) braahma.nam upanayiita(upa-nii-)  
Va.11.50 garbhaat ekaada"[se.su](http://se.su) raajanyam //  
Va.11.51 garbhaat dvaada"[se.su](http://se.su) vai"syam //  
Va.11.52 paalaa"sas vaa da.n.das braahma.nasya //  
Va.11.53 naiyyagrodhas k.satriyasya vaa //  
Va.11.54 audumbaras vaa vai"syasya //  

  
  
Brāhmaṇa to hold palāśa.  

Kṣatriya nyagrodha  

Vaiśya udumbara  

========  
  

Manusṃṛti (२:४५-४६) reads:  
  

ब्राह्मणो बैल्वपालाशौ क्षत्रियो वाटखादिरौ।  

पैलवौदुम्बरौ वैश्यो दण्डानर्हन्ति धर्मतः।।  

केशान्तिको ब्राह्मणस्य दण्डः कार्यः प्रमाणतः।  

ललटसंमितो राज्ञः स्यात्तु नासान्तिको विशः।।  

ब्राह्मणस्य = बिल्वं पलाशम्।  
क्षत्रियस्य = वट खादिर।  

वैश्यस्य = पीलु उदुम्बर ।  
  

भवदीयः
