इन्द्रियस्यैवसंस्कारः शब्दस्यैवॊभयस्य वा /

क्रियतॆ ध्वनिभिर् वादास् त्रयॊ ऽभिव्यक्तिवादिनाम् // वाक्य\_१।८० //

इन्द्रियस्यैव संस्कारः समाधानाञ्जनादिभिः /

विषयस्य तु संस्कारः तद्गन्धप्रतिपत्तयॆ // वाक्य\_१।८१ //

चक्षुषः प्राप्यकारित्वॆ तॆजसा तु द्वयॊर् अपि /

विषयॆन्द्रिययॊर् इष्टा संस्कारः स क्रमॊ ध्वनॆः // वाक्य\_१।८२ //

स्फॊटरूपाविभागॆन ध्वनॆर् ग्रहणम् इष्यतॆ /

कैश् चित् ध्वनिर् असंवॆद्यः स्वतन्त्रॊ ऽन्यैः प्रकल्पितः // वाक्य\_१।८३ //

यथानुवाकः श्लॊकॊ वा सॊढत्वम् उपगच्छतॆ /

आवृत्त्या न तु स ग्रन्थः प्रत्यावृत्ति निरूप्यतॆ // वाक्य\_१।८४ //

प्रत्ययैर् अनुपाख्यॆयैर् ग्रहणानुगुणैस् तथा /

ध्वनिप्रकाशितॆ शब्दॆ स्वरूपम् अवधार्यतॆ // वाक्य\_१।८५ //

नादैर् आहितबीजायाम् अन्त्यॆन ध्वनिना सह /

आवृत्तपरिपाकायां बुद्धौ शब्दॊ ऽवधार्यतॆ // वाक्य\_१।८६ //
