सत्यमुक्तं महाबलभट्टाः! सन्त्यनेकानि रामायणानि बहुभिः प्रणीतानि, येषां भारेण भूमेर्भरं सोढुमशक्नुवानः फणिपतिरपि स्वशिरःकम्पमनुभूतवानिति अस्मदीयस्य तोरवे नरसिंहस्योक्तिः।

  

इदानीं कम्बरामायणानुसारेण दशरथस्य नाम नेमिरित्यासीदिति ज्ञायते (कम्बरामायणे युद्धकाण्डे) इति विज्ञाप्यते\|

  

एवं पुराणान्यपि स्रोतांस्येव विज्ञानस्य। भवन्मित्रमभिनन्दनान्यर्हति। मम सादरं प्रणामास्तेभो विज्ञाप्यन्ताम्। तदुक्तेः प्रमाणं तु मत्स्यपुराणे ४८.

  
atha dharmarathasyābhūtputraścitrarathaḥ kila / (94.1)  
tasya satyarathaḥ putrastasmād***daśarathaḥ*** kila // (94.2)  
lomapāda iti khyātastasya śāntā sutābhavat / (95.1)

  

द्रव्यष्टव्यम्। भवभूतेस्तु पुराणान्तरं वा प्रमाणं मृग्यम्। अथवा उभयोरभेदेन कविस्वातन्त्र्येण तथा कल्पितवान् स्यात्। रामायणे बालकाण्डे चैवं दृश्यते -

  

rājā daśaratho nāmnā śrīmān satyapratiśravaḥ // (2.2)

aṅgarājena sakhyaṃ ca tasya rājño bhaviṣyati / (3.1)

kanyā cāsya mahābhāgā śāntā nāma bhaviṣyati // (3.2)

putras tv aṅgasya rājñas tu romapāda iti śrutaḥ / (4.1)

taṃ sa rājā daśaratho gamiṣyati mahāyaśāḥ // (4.2)

Rām, Bā, 10, 2-4)

  

उपर्युद्धृतभागे दशरथस्यैव शान्ता कन्या इति अभूहः कर्तुं शक्यते। यदाधारेण भवभूतेः कथनं संगतमिव भाति। रोमपादाय तस्या दानं विना न संगच्छते च। अत्र प्रमाणान्तरं वर्तते वा इति मृग्यम्।

  

शेषे शेषा विद्वांसः प्रमाणम्।

  

ह्यः मया केवलं विनोदार्थमुक्तं क्षम्यताम्।

  

भवदीयः।
