एतदेव प्रथममुक्तं चेनासीदेव प्रश्नस्यावकाशः।

  

तच्चक्षुर्देवहितं शुक्रमुच्चरत् ।

पश्येम शरदः शतं जीवेम शरदः शतम् ॥१६॥

  

एषा तु वैदिकी प्रार्थना, किंच,\]

  

शतमानं भवति । शतायुः पुरुषः शतेन्द्रिय आयुष्येवेन्द्रिये प्रति तिष्ठति ।

  

इति च श्रूयते। आकरं तु न जानामि।

  

मत्स्यपुराणे १४५तमेऽध्याये -

  

सूत उवाच / (१.१)

मन्वन्तराणि यानि स्युः कल्पे कल्पे चतुर्दश / (१.२)

व्यतीतानागतानि स्युर्यानि मन्वन्तरेष्विह // (१.३)

विस्तरेणानुपूर्व्याच्च स्थितिं वक्ष्ये युगे युगे / (२.१)

तस्मिन्युगे च सम्भूतिर्यासां यावच्च जीवितम् // (२.२)

युगमात्रं तु जीवन्ति न्यूनं तत्स्याद्द्वयेन च / (३.१)

चतुर्दशसु तावन्तो ज्ञेया मन्वन्तरेष्विह // (३.२)

मनुष्याणां पशूनां च पक्षिणां स्थावरैः सह / (४.१)

तेषामायुरुपक्रान्तं युगधर्मेषु सर्वशः // (४.२)

तथैवायुः परिक्रान्तं युगधर्मेषु सर्वशः / (५.१)

अस्थितिं च कलौ दृष्ट्वा भूतानामायुषश्च वै // (५.२)

परमायुः शतं त्वेतन्मानुषाणां कलौ स्मृतम् / (६.१)

  

इत्युक्तम्। देवानां मानुषाणां पक्षिणां च पृथक् पृथग् गणना। इति तत एवावगम्यते।
