शब्दैरेभि: सभाजयितुकामा: श्रीनिवासवरखेडीमहाशयान !  
श्रीशः प्रसीदतितमां तमसां निहन्ता ज्योतिः स्फुरत्यविरतं वत संस्कृतश्री: /  
श्रीमन्निवासवरखेटवरेण्यबन्धुर्वाणीविलासविदथे'थ निदेशको'स्ति //  
कर्णाटके संस्कृतकर्णपूर्णे नभोगुणे संस्कृतदिव्यवर्ने /  
भूयात्सदा भासुरसंस्कृताध्वा राष्ट्रे ह राष्ट्री जयतीह धन्या //  
राज्यालि-भाषा भवताद द्वितीया राष्ट्रस्य भाषा भवताद द्वितीया /  
सा नो'द्वितीया जगतां शरण्या भूयो भवेद्भारतभूतिसेतु: //  
वर्धापयामो विदुषां वदातं मोमुद्यमाना अभिनन्दनानि /  
विज्नापयामो वत सादरं ये वशंवदा भारतभारतीद्धा: //  
सुरेन्द्रमोहनमिश्र:
