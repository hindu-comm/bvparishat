आंग्लं प्रथमभाषेति लिलेखिषव 'इण्डियन्स' ते बहव: /  
जनगणने तु संस्कृतं पुरस्कुर्वन्तु भारतीया देवा: //  
दशमप्रश्ने स्थाने मातृभाषाया लिखन्तु संस्कृतं /  
गङ्गा सागरसंगममवाप्य किं गङ्गात्वं जहाति ?//  
भारतीयभाषा हि तथा संस्कृतगङ्गाया विवर्ता: /  
विद्वांसो विमत्सरा बुद्ध्वा कुर्युः कर्तव्यं समये //  
मेकालेसदृशास्ते निपुणा:समर्पिताः स्वलक्ष्यपूर्तौ /  
हे भारत ! जागृहि रे ! मर्यादाप्रश्नः स एषः //  
" सक्ताः कर्मण्यविद्वांसो यथा कुर्वन्ति भारत ! /  
 कुर्याद्विद्वाम्स्तथासक्तश्चिकीर्षुर्लोकसंग्रहं // "  
यदि त्वं वर्तसे नैव जातु कर्मण्यतन्द्रितः /  
तव वर्त्मा'नुवर्तन्ते मनुष्याः पार्थ सर्वशः //  
अल्पबुद्धिः संशयी न स श्रोतव्यः काले विषमे हि /  
विश्वे जागृयाम ते वयं पथिकृतः संस्कृतसंस्कारैः//  
कार्पण्यदोषान्पुरतो विसृज्य देवर्षिपितृस्वमिदं विभाव्य /  
तद्रक्षणे विश्वहिताय मित्र ! वाणी सुरक्ष्या खलु भारतात्मा !//  
विदुषां विधेयः  
सुरेन्द्रमोहनमिश्रः  
कुरुक्षेत्रम् ।
