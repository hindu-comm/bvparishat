‘पुनः पुनः लपति’ इति लालप्यते। लालप्यते = लप् + यङ् + त। ‘लालप्यते’ इति लालप्यमानः। तस्य लालप्यमानस्य। लालप्यमानस्य = लप् + यङ् + शानच् + ङस्।

लपँ व्यक्तायां वाचि (धा॰पा॰ ४०२) → लप् → धातोरेकाचो हलादेः क्रियासमभिहारे यङ् (पा॰सू॰ ३.१.२२) → लप् यङ् → लप् य → सन्यङोः (पा॰सू॰ ६.१.९) → लप् लप् य → अभ्यासकार्यम् → हलादिः शेषः (पा॰सू॰ ७.४.६०) → ल लप् य → दीर्घोऽकितः (पा॰सू॰ ७.४.८३) → ला लप् य → लालप्य → सनाद्यन्ता धातवः (पा॰सू॰ ३.१.३२) → धातु सञ्ज्ञा → अनुदात्तङित आत्मनेपदम् (पा॰सू॰ १.३.१२) → वर्तमाने लट् (पा॰सू॰ ३.२.१२३) → लालप्य लट् → लालप्य त → कर्तरि शप्‌ (पा॰सू॰ ३.१.६८) → लालप्य शप् त → लालप्य अ त → अतो गुणे (पा॰सू॰ ६.१.९७) → लालप्य त → टित आत्मनेपदानां टेरे (पा॰सू॰ ३.४.७९) → लालप्य ते → लालप्यते।

सनाद्यन्ता धातवः (पा॰सू॰ ३.१.३२) → धातु सञ्ज्ञा → अनुदात्तङित आत्मनेपदम् (पा॰सू॰ १.३.१२) → वर्तमाने लट् (पा॰सू॰ ३.२.१२३) → लालप्य लट् → लटः शतृशानचावप्रथमा समानाधिकरणे (पा॰सू॰ ३.२.१२४) → लालप्य शानच् → लालप्य आन → कर्तरि शप्‌ (पा॰सू॰ ३.१.६८) → लालप्य शप् आन → लालप्य अ आन → अतो गुणे (पा॰सू॰ ६.१.९७) → लालप्य आन → आने मुक् (पा॰सू॰ ७.२.८२) → आद्यन्तौ टकितौ (पा॰सू॰ १.१.४६) → लालप्य मुँक् आन → लालप्य म् आन → लालप्यमान → कृत्तद्धितसमासाश्च (पा॰सू॰ १.२.४६) → प्रातिपदिकसञ्ज्ञा → विभक्तिकार्यम् → लालप्यमान ङस् → टाङसिङसामिनात्स्याः (पा॰सू॰ ७.१.१२) → अनेकाल्शित्सर्वस्य (पा॰सू॰ १.१.५५) → लालप्यमान स्य → लालप्यमानस्य।
