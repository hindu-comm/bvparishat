Bhartrihari is more specific in his details. Please observe the names of books, persons and places clearlyhighlighted in the verses below. In spite of all such historical details, one highly learned Indologist attributes Bhartrihari's crediting the ideas in his book to his predecessors as lack of historical consciousness. He pities Bhartrihari for not owning up his innovations that make vAkyapadIyam.



प्रायॆण संक्षॆपरुचीन् अल्पविद्यापरिग्रहान् /

संप्राप्य वैयाकरणान् ***संग्रहॆ*** ऽस्तम् उपागतॆ // वाक्य\_२।४८१ //

कृतॆ ऽथ ***पतञ्जलिना*** गुरुणा तीर्थदर्शिना /

सर्वॆसं न्यायबीजानां महाभाष्यॆ निबन्धनॆ // वाक्य\_२।४८२ //

अलब्धगाधॆ गाम्भीर्याद् उत्तान इव सौष्ठवात् /

तस्मिन्न् अकृतबुद्धीनाम् नैवावास्थित निश्चयः // वाक्य\_२।४८३ //

***वैजिसौभवहर्यक्षैः*** शुष्कतर्कानुसारिभिः /

आर्षॆ विप्लावितॆ ग्रन्थॆ संग्रहप्रतिकञ्चुकॆ // वाक्य\_२।४८४ //

यः ***पतञ्जलिशिष्यॆभ्यॊ*** भ्रष्टॊ व्याकरणागमः /

कालॆन ***दाक्षिणात्यॆषु*** ग्रन्थमात्रॊ व्यवस्थितः // वाक्य\_२।४८५ //

***पर्वताद्*** आगमं लब्ध्वा भाष्यबीजानुसारिभिः /

स नीतॊ बहुशाखत्वं ***चान्द्राचार्यादिभिः*** पुनः // वाक्य\_२।४८६ //

न्यायप्रस्थानमार्गांस् तान् अभ्यस्य स्वम् च दर्शनम् /

प्रणीतॊ ***गुरुणास्माकम्*** अयम् आगमसंग्रहः // वाक्य\_२।४८७ //
