At the occasion of the World Sanskrit Conference in Bangkok, a public distribution of theSanskrit Heritage Engine software has been released,

together with a user's reference manual for the various tools.

This version allows using the tools as Web services on your own workstation, provided it runs some version of Unix, such as Linux or Mac OSX.

The[user manual](http://sanskrit.inria.fr/manual.html)tells how to download the software and install it. Please report any difficulty or anomaly.
