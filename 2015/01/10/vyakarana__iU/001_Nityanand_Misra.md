प्रकर्षेण जानातीति प्राज्ञी।

प्र ज्ञा अवबोधने (धा॰पा॰ १५०७) → प्र ज्ञा → आतश्चोपसर्गे (पा॰सू॰ ३.१.१३६) → प्र ज्ञा क → प्र ज्ञा अ → आतो लोप इटि च (पा॰सू॰ ६.४.६४) → प्र ज्ञ् अ → प्रज्ञ → प्रज्ञादिभ्यश्च (पा॰सू॰ ५.४.३८) → स्वार्थेऽण् → प्रज्ञ अण् → प्रज्ञ अ → यचि भम् (पा॰सू॰ १.४.१८) → भसञ्ज्ञा → तद्धितेष्वचामादेः (पा॰सू॰ ७.२.११७) → प्राज्ञ अ → यस्येति च (पा॰सू॰ ६.४.१४८) → प्राज्ञ् अ → प्राज्ञ → टिड्ढाणञ्द्वयसज्दघ्नञ्मात्रच्तयप्ठक्ठञ्कञ्क्वरपः (पा॰सू॰ ४.१.१५) → प्राज्ञ ङीप् → प्राज्ञ ई → यचि भम् (पा॰सू॰ १.४.१८) → भसञ्ज्ञा → यस्येति च (पा॰सू॰ ६.४.१४८) → प्राज्ञ् ई → प्राज्ञी → ङ्याप्प्रातिपदिकात्‌ (पा॰सू॰ ४.१.१) → स्वौजसमौट्छष्टाभ्याम्भिस्ङेभ्याम्भ्यस्ङसिभ्याम्भ्यस्ङसोसाम्ङ्योस्सुप्‌ (पा॰सू॰ ४.१.२) → प्राज्ञी सुँ → प्राज्ञी स् → हल्ङ्याब्भ्यो दीर्घात्सुतिस्यपृक्तं हल् (पा॰सू॰ ६.१.६८) → सोर्लोपः → प्राज्ञी।
