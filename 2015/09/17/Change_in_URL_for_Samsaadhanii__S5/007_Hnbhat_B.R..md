What about this formation?

  

  

|           |             |           |        |                   | |-----------|-------------|-----------|--------|-------------------| | प्रथमपदम् | द्वितीयपदम् | समस्तपदम् | सन्धिः | सूत्रम्/वार्तिकम् |

|     |     |       | |-----|-----|-------| | हरी | इमौ | हरीमौ |

|                   |                             | |-------------------|-----------------------------| | सवर्णदीर्घ-सन्धिः | अकः सवर्णे दीर्घः (6।1।101) |

|| || ||

  

  

The above case is not the correct form. But it will not have any Sandhi, as perईदूदेद्विवचनं प्रगृह्यम्॥ 1.1.11 it will not undergo any change.

  

  

The same with this example:

  

  

|           |             |           |        |                   | |-----------|-------------|-----------|--------|-------------------| | प्रथमपदम् | द्वितीयपदम् | समस्तपदम् | सन्धिः | सूत्रम्/वार्तिकम् |

|        |     |            |                   |                                          | |--------|-----|------------|-------------------|------------------------------------------| | विष्णू | इमौ | विष्ण्विमौ | यण्-सन्धिः        | इको यणचि (6।1।77)                        | | विष्णू | इमौ | विष्णु इमौ | प्रकृतिभाव-सन्धिः | इकोऽसवर्णे शाकल्यस्य ह्रस्वश्च (6।1।127) |

  

  

Here the first is wrong as the above and in the second there is no such समास with विष्णु इमौ as explained in the title.

  

In other words, the software takes by Sandhi, where there is change of form and not otherwise, considered as प्रकृतिभाव as in the above case.
