As you guessed, the word वावल्ल can be derived in the Paninian system from the यङन्त root वावल्ल्य or the यङ्लुगन्त root वावल्ल्, both originally from वल्लँ संवरणे सञ्चरणे च (धा.पा. ४९२). Similarly the word मामल्ल can be derived in the Paninian system from the यङन्त root मामल्ल्य or the यङ्लुगन्त root मामल्ल्, both originally from मल्लँ धारणे (धा.पा. ४९४).  

वावल्ल्यते/वावल्ल्ति/वावल्लीति (=पुनः पुनर्वल्लते भृशं वा वल्लते कुटिलं वा वल्लते) इति वावल्लः। वावल्ल = वल्ल् + यङ्/यङ्लुक् + अच्।मामल्ल्यते/मामल्ल्ति/मामल्लीति (=पुनः पुनर्मल्लते भृशं वा मल्लते) इति मामल्लः। मामल्ल = मल्ल् + यङ्/यङ्लुक् + अच्।

  

Even if the lexicons do not list them, the wordsवावल्लandमामल्लare notअपशब्दs. They can very well be derived in the Paninian system, and that too without going intoपृषोदरादिor considering them asदेशिपदs as Dr Korada suggests.

  

I show derivations for वावल्ल, those for मामल्ल are similar. The only difference is that for वावल्ल, the rule ordaining यङ् is धातोरेकाचो हलादेः क्रियासमभिहारे यङ् (पा॰सू॰ ३.१.२२) when the meaning is संवरण while it is नित्यं कौटिल्ये गतौ (पा॰सू॰ ३.१.२३) when the meaning is सञ्चरण. For मामल्ल the rule ordaining यङ् is धातोरेकाचो हलादेः क्रियासमभिहारे यङ् (पा॰सू॰ ३.१.२२) only.

  

The यङन्त root वावल्ल्य is derived as वल्लँ संवरणे सञ्चरणे च (धा.पा. ४९२) → अनुबन्धलोपः → वल्ल् → धातोरेकाचो हलादेः क्रियासमभिहारे यङ् (पा॰सू॰ ३.१.२२)/नित्यं कौटिल्ये गतौ (पा॰सू॰ ३.१.२३) → वल्ल् यङ् → अनुबन्धलोपः → वल्ल् य → सन्यङोः (पा॰सू॰ ६.१.९) → वल्ल् वल्ल् य → अभ्यासकार्यम् → हलादिः शेषः (पा॰सू॰ ७.४.६०) → व वल्ल् य → दीर्घोऽकितः (पा॰सू॰ ७.४.८३) → वा वल्ल् य → वावल्ल्य → सनाद्यन्ता धातवः (पा॰सू॰ ३.१.३२) → धातुसञ्ज्ञा।

  

वावल्ल is now derived as वावल्ल्य → अज्विधिः सर्वधातुभ्यः (वा॰ ३.१.१३४) → वावल्ल्य + अच् → अनुबन्धलोपः → वावल्ल्य + अ → यङोऽचि च (पा॰सू॰ २.४.७४) → वावल्ल् अ → वावल्ल → विभक्तिकार्यम् → वावल्लः।

  

The यङ्लुगन्त root वावल्ल् is derived as वल्लँ संवरणे सञ्चरणे च (धा.पा. ४९२) → अनुबन्धलोपः → वल्ल् → नित्यं कौटिल्ये गतौ (पा॰सू॰ ३.१.२३) → वल्ल् यङ् → अनुबन्धलोपः → वल्ल् य → यङोऽचि च (पा॰सू॰ २.४.७४) → वल्ल् → प्रत्ययलोपे प्रत्ययलक्षणम् (पा॰सू॰ १.१.६२) → सन्यङोः (पा॰सू॰ ६.१.९) → वल्ल् वल्ल् → अभ्यासकार्यम् → हलादिः शेषः (पा॰सू॰ ७.४.६०) → व वल्ल् → दीर्घोऽकितः (पा॰सू॰ ७.४.८३) → वा वल्ल् → वावल्ल् → सनाद्यन्ता धातवः (पा॰सू॰ ३.१.३२) → धातुसञ्ज्ञा।

  

वावल्ल is now derived as वावल्ल् → अज्विधिः सर्वधातुभ्यः (वा॰ ३.१.१३४) → वावल्ल् + अच् → अनुबन्धलोपः → वावल्ल् + अ → वावल्ल → विभक्तिकार्यम् → वावल्लः।

  

The derivations of the conjugations वावल्ल्यते/ वावल्ल्ति/वावल्लीति are not provided. If the OP is interested, he may please contact me off the list.
