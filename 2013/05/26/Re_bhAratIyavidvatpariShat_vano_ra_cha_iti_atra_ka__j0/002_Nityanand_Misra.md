The question is not stated very cleraly, but I think what Subhendu Mahodaya meant to ask is that if वनः is in पञ्चमी, then how come an आदेश (अन्तादेश) is ordained, since आदेश is usually ordained with षष्ठी (e.g. युवोरनाकौ).

  

If this is indeed the quesion, then the answer to this is that the Sutra has two sentences (वाक्यभेद). The first sentence is वनो ङीप् (with ङीप् carried from 4-1-5 ऋन्नेभ्यो ङीप्‌) where वनः is पञ्चमी form and ङीप् is ordained. The second sentence is वनो र च where वनः is in षष्ठी form and by using the परिभाषा 1-1-52 अलोऽन्त्यस्य, only the end of वन् (i.e. न्) is replaced by र्.

  

For more on this, please refer the commentary by Ram Nath Sharma (English) and Prathama Vritti by Brahmadatta Jijnasu (Hindi) where this is explained. Both of them have marked वनः as षष्ठी (6/1 by Sharma, ६।१ by Jijnasu) which is as per the second वाक्य.
