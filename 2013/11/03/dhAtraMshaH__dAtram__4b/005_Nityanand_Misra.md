Let me evaluate the original argument first. I will use "=>" for the implication relationship in logic, as in (x \> 5) => (x \> 2) for real x. As I understand, the argument

"द्वित्वनिषेधे तु स्थानिवद्भावः, तेन दीर्घादाचार्याणाम् इति सूत्रस्य प्रवृत्तिर्न भवति"

means

1) When there is द्वित्वनिषेध, there is स्थानिवद्भाव (द्वित्वनिषेध => स्थानिवद्भाव)

2) As there is स्थानिवद्भाव, the rule दीर्घादाचार्याणाम् does not apply. (स्थानिवद्भाव => दीर्घादाचार्याणाम् does not apply)

The two above can be written as

(A) द्वित्वनिषेध => स्थानिवद्भाव => दीर्घादाचार्याणाम् does not apply

Now the first step in the argument assumes द्वित्वनिषेध. Where does द्वित्वनिषेध in this context come from? From the rule दीर्घादाचार्याणाम्. So we have

(B) दीर्घादाचार्याणाम् applies => द्वित्वनिषेध

From (A) and (B), we have

(C) दीर्घादाचार्याणाम् applies => द्वित्वनिषेध => स्थानिवद्भाव => दीर्घादाचार्याणाम् does not apply

As "implication" is a "transitive relation", i.e. \[(A => B) and (B => C)\] => (A => C),and we have

(D) दीर्घादाचार्याणाम् applies => दीर्घादाचार्याणाम् does not apply

This is nothing but *reductio ad absurdum*, and so the original assumption must be false. I am not aware if this is on purpose, you may want to check with your friend. As for me, if I agreed with all the steps in the argument, I would have found the argument useful. But I do not agree with the first step (द्वित्वनिषेध => स्थानिवद्भाव).

The question as I see from your second mail is whether 1-1-58 न पदान्तद्विर्वचनवरेयलोपस्वरसवर्णानुस्वारदीर्घजश्चर्विधिषु (exceptions to 1-1-57) means a निषेधसूत्र is to treat an आदेश as स्थानिवत्. IMHO the answer is no, the reason beingthe Anuvritti of पूर्वविधौ from 1-1-57 in 1-1-58. With the Anuvrittis, the Sutras read -

1-1-57 अचः परस्मिन्पूर्वविधौ स्थानिवत् आदेशः

1-1-58 न पदान्तद्विर्वचनवरेयलोपस्वरसवर्णानुस्वारदीर्घजश्चर्विधिषु स्थानिवत् आदेशः अचः परस्मिन्पूर्वविधौ

1-1-58 lists exceptions to 1-1-57. The domain of 1-1-57 is not all Sutras, but it is clearly limited to पूर्वविधि Sutras. Nothing is said about निषेधसूत्रs (of any विधि) either in the general rule 1-1-57 or in the exception rule 1-1-58. So, even if निषेधसूत्र is not explicitly exempted by 1-1-58, one cannot assume either way - whether a निषेधसूत्र is to treat or not to consider an आदेश as स्थानिवत्.

One can think of 1-1-57 as (Order A): "पुरुषमानय", and 1-1-58 as (Order B): "न ब्राह्मणम्". The second order is a प्रसज्य प्रतिषेध when seen in itself. But with Anuvritti of "पुरुषमानय", the second order becomes "पुरुषमानय ब्राह्मणं न", i.e., "अब्राह्मणं पुरुषमानय" which is a पर्युदास प्रतिषेध. It cannot be concluded that the second order says न ब्राह्मणम्, so a horse can be brought. Only a पुरुष is to be brought as the first order limits the scope. Similarly 1-1-58 says न पदान्त..., but it cannot be concluded that for a द्वित्वनिषेध there will be स्थानिवद्भाव since both 1-1-57 and 1-1-58 pertain to only पूर्वविधि Sutras but notनिषेधसूत्रs.

This is my thought process. I will be happy to be corrected if I am wrong. One may benefit by seeing what commentaries say on 1-1-58 - do they treat the exceptions asप्रसज्यप्रतिषेधorपर्युदासप्रतिषेध.
