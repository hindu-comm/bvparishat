On second thoughts, application of नस्तद्धिते \[अङ्गस्य भस्य टेः लोपः\] (6-4-144) is blocked in this case by अन् \[अङ्गस्य भस्य प्रकृत्या अणि\] (6-4-167). I was misled by entries for सौपर्व in Monier Williams and a Hindi dictionary which (incorrectly?) say it is from सुपर्वन्.  
  
With 6-4-144 not being applicable, the form should be सौपर्वणी as below  
  
सुपर्वन् + अण् -> तस्य लोपः \[इत्\] (1-3-9) -> सुपर्वन् अ -> तद्धितेष्वचामादेः \[अङ्गस्य वृद्धिः अचो ञ्णिति\] (7-2-117) -> सौपर्वन् अ -> सौपर्वन -> अट्कुप्वाङ्नुम्व्यवायेऽपि \[रषाभ्यां नो णः समानपदे\] (8-4-2) -> सौपर्वण  
सौपर्वण ङीप् -> सौपर्वण ई -> यस्येति च \[अङ्गस्य भस्य लोपः तद्धिते\] (6-4-148) -> सौपर्वण् ई -> सौपर्वणी  
  
सौपर्वाणी cannot result in any case, whether 6-4-144 is applicable or not.
