Anumapa Mahodaye

  

दौहित्रायण is what you are looking for. Listed in Vachaspatyam and also by V S Apte, Monier Williams (incorrectly as दौहित्रायाण in the digital version),Otto Böhtlingk, et cetera.

  

The Prakriya is दुहितुर्गोत्रापत्यं युवा = दुहितृ + अञ् + फक् = दौहित्रायण by considering it inहरितादिगण (refer हरितादिभ्योऽञः 4-1-100).

  

The broad derivation is as below (विस्तारभिया सङ्क्षिप्यते) -

  

दुहितृ -> अनृष्यानन्तर्ये बिदाऽदिभ्योऽञ् (4-1-104) -> दुहितृ अञ् -> अनुबन्धलोप -> दुहितृ अ -> तद्धितेष्वचामादेः (7-2-117) -> आदिवृद्धि -> दौहितृ अ -> इको यणचि (6-1-77) -> दौहित्र -> हरितादिभ्योऽञः (4-1-100) -> दौहित्र फक् -> अनुबन्धलोप -> दौहित्र फ -> आयनेयीनीयियः फढखच्छघां प्रत्ययादीनाम्‌ (7-1-2) -> दौहित्र आयन -> अकः सवर्णे दीर्घः (6-1-101) -> दौहित्रायन -> 8-4-2 अट्कुप्वाङ्नुम्व्यवायेऽपि -> दौहित्रायण -> विभक्तिकार्य -> दौहित्रायणः
