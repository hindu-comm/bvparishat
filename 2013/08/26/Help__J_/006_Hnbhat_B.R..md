अन्यच् च-

अयं निजः परो वेति गणना लघु-चेतसाम् \|

उदार-चरितानां तु वसुधैव कुटुम्बकम् // १।७१ //

  

The number may differ in different editions of Hitoadesha. But it quotes from many other sources without mentioning the source as

  

सहसा विदधीत न क्रियाम्

अविवेकः परमापदां पदम् \|

वृणते हि विमृश्य कारिणं

गुण-लुब्धाः स्वयम् एव सम्पदः // ४।१०४ //

  

the above being from

  

सहसा विदधीत न क्रियाम् अविवेकः परम् आपदां पदम् /

वृणते हि विमृश्यकारिणं गुणलुब्धाः स्वयम् एव सम्पदः // कितातार्जुनीयं २।३० //

  

as we know. Others too are from different sources according to the scholars who have edited the text.
