Multiple affixes (तृच्/ण्वुल्/ अच्) are ordained by Panini in the sense of agent (कर्तरि). To get the meaning of "प्रकृष्टं वक्ति" in कर्ता, प्रवक्ता is only one of the several possibilities – I can think of at least two more (there could be even more)

1.  With suffix तृच् one gets प्रवक्ता - प्र वच् -> 3-1-133 ण्वुल्तृचौ
    -> प्र वच् तृच् -> 1-3-3 हलन्त्यम्, 1-3-9 तस्य लोपः -> अनुबन्धलोप ->
    प्र वच् तृ -> 8-2-30 चोः कुः -> कुत्व -> प्र वक् तृ -> प्रवक्तृ ->
    विभक्तिकार्य -> प्रवक्ता  
2.  With suffix ण्वुल् one gets प्रवाचकः - प्र वच् -> 3-1-133 ण्वुल्तृचौ
    -> प्र वच् ण्वुल् -> 1-3-3 हलन्त्यम्, 1-3-7 चुटू, 1-3-9 तस्य लोपः
    -> अनुबन्धलोप -> प्र वच् वु -> 7-2-115 अचो ञ्णिति -> वृद्धि -> प्र
    वाच् वु -> 7-1-1 युवोरनाकौ -> प्र वाच् अक -> प्रवाचक -> विभक्तिकार्य
    -> प्रवाचकः  
3.  With suffix अच् one gets प्रवचः - प्र वच् -> 3-1-134
    नन्दिग्रहिपचादिभ्यो ल्युणिन्यचः -> पचाद्यच् -> प्र वच् अच् -> 1-3-3
    हलन्त्यम्, 1-3-9 तस्य लोपः -> अनुबन्धलोप -> प्र वच् अ -> प्रवच ->
    विभक्तिकार्य -> प्रवचः  

While both प्रवक्तृ and प्रवाचक are listed in Koshas including Monier-Williams, use of पचाद्यच् from वच् is supported by Kashika which explicitly lists the root वच in the पचादि आकृतिगण, and also by the maxim अज्विधिः सर्वधातुभ्यः (Ishvarachandra lists अज्विधिः सर्वधातुभ्यः as a Varttika under 3-1-134 in the Chandralekha commentary on Ashtadhyayi).

  

Moreover, वच (=वच्+अच्) is listed in both masculine (कर्तरि अच्) and neuter (भावे अच्) in Koshas, including Apte and Monier-Williams. Therefore, unless there is an explicit rule/Varttika/Nyaya which blocks अच् from प्र + वच् (thus overriding पचाद्यच् and अज्विधिः सर्वधातुभ्यः), I find it difficult to admit that प्रवचिने is असाधु. In absence of no such blocking provision, प्र + वच् + भावे अच् + प्राशस्त्ये इनि + ङे will definitely result in प्रवचिने as per Paninian Prakriya below (excuse mistakes, if any) –

प्र वच् + भावे अच् -> प्रवच (as above) -> 5-2-115 अत इनिठनौ -> प्रवच इनि -> 1-3-2 उपदेशेऽजनुनासिक इत्, 1-3-9 तस्य लोपः -> अनुबन्धलोप -> प्रवच इन् -> 1-4-18 यचि भम् -> "प्रवच" इत्यस्य भसञ्ज्ञा -> 6-4-148 यस्येति च -> अलोप -> प्रवच् इन् -> प्रवचिन् -> 4-1-2 स्वौजसमौट्छष्टाभ्याम्भिस्ङेभ्याम्भ्यस्ङसिभ्याम्भ्यस्ङसोसाम्ङ्योस्सुप्‌ -> प्रवचिन् ङे -> 1-3-3 हलन्त्यम्, 1-3-9 तस्य लोपः -> अनुबन्धलोप -> प्रवचिन् ए -> प्रवचिने इति रूपं सिद्धम्

My understanding is that ताच्छीलिकाः are not a subset of रूढिशब्दाः, but similar to रूढिशब्दाः. कैयट in प्रदीप explains प्रकार in रूढिशब्दप्रकाराः as सादृश्य. Both कैयट and नागेश comment that while no रूढिशब्द are preceded by गतिs, some ताच्छीलिकs are and some are not, and so my conclusion is that “some ताच्छीलिकाः are like रूढिशब्दाः, some are not”. Who are we to say if ताच्छील्ये णिनिः is possible or not here (के वयं वराकाः)? But if “प्रवाचिने” is seen in a शिष्टप्रयोग, ताच्छील्ये णिनिः is one of the ways to explain it.
