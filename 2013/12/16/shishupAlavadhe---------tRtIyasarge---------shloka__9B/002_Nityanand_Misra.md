The Prakriya cannot be given since contrary to what Narasimhacharya Ji says, अध्यूष् is not the प्रातिपदिक here at all! Neither is the प्रातिपदिक अध्यूष्, nor is it षकारान्त!

अध्यूषुषः is actually declined form of अध्यूषिवस् which is a क्वसन्त and सकारान्त प्रातिपदिक. अध्यूषिवस् -> पुल्लिङ्ग, षष्ठीविभक्ति, प्रपु एव -> अध्यूषुषः

Here is the derivation of अध्यूषिवस्

अधि वस् -> 3-2-115 परोक्षे लिँट् -> अधि वस् लिँट् -> 3-2-108 भाषायां सदवसश्रुवः -> अधि वस् क्वसुँ -> 1-3-2 उपदेशेऽजनुनासिक इत्, 1-3-8 लशक्वतद्धिते, 1-3-9 तस्य लोपः -> अनुबन्धलोपः -> अधि वस् वस् -> अधि व् अस् वस् -> 6-1-15 वचिस्वपियजादीनां किति -> सम्प्रसारणम् -> अधि उ अस् वस् -> 6-1-108 सम्प्रसारणाच्च -> पूर्वरूपैकादेशः -> अधि उस् वस् -> 6-1-8 लिटि धातोरनभ्यासस्य -> अधि उस् उस् वस् -> 7-4-60 हलादिः शेषः -> अधि उ उस् वस् -> 6-1-101 अकः सवर्णे दीर्घः -> अधि ऊस् वस् -> 7-2-35 आर्धधातुकस्येड् वलादेः ->इडागमः -> 7-2-10 एकाच उपदेशेऽनुदात्तात् -> इडागमनिषेधः -> 7-2-13 कृसृभृवृस्तुद्रुस्रुश्रुवो लिटि -> इडागमनिषेधाभावः -> इडागमः -> 1-1-46 आद्यन्तौ टकितौ -> आद्यागमः -> अधि ऊस् इट् वस् -> 1-3-3 हलन्त्यम्, 1-3-9 तस्य लोपः -> अनुबन्धलोपः -> अधि ऊस् इ वस् -> 8-3-60 शासिवसिघसीनां च -> षत्वम् -> अधि ऊष् इ वस् -> 6-1-77 इको यणचि -> 1-3-10 यथासंख्यमनुदेशः समानाम् -> अध्य् ऊष् इ वस् -> अध्यूषिवस्

Here is the derivation of अध्यूषुषः

अध्यूषिवस् ङस् -> 1-3-8 लशक्वतद्धिते, , 1-3-9 तस्य लोपः -> अनुबन्धलोपः -> अध्यूषिवस् अस् -> अध्यूषि वस् अस् -> 1-4-13 यस्मात्प्रत्ययविधिस्तदादि प्रत्ययेऽङ्गम् -> अङ्गसंज्ञा -> 1-4-18 यचि भम् -> भसंज्ञा -> 6-4-131 वसोः सम्प्रसारणम् -> अध्यूषि उ अस् अस् -> अकृतव्यूहाः पाणिनीयाः -> सम्प्रसारण इडागमनिवर्तनम् -> अध्यूष् उ अस् अस् -> 6-1-108 सम्प्रसारणाच्च -> पूर्वरूपैकादेशः -> अध्यूष् उस् अस् -> अध्यूषु स् अस् -> 8-3-59 आदेशप्रत्यययोः -> षत्वम् -> अध्यूषुष् अस् -> अध्यूषुषस् -> 8-2-66 ससजुषो रुः -> अध्यूषुषरु -> 8-3-15 खरवसानयोर्विसर्जनीयः -> अध्यूषुषः
