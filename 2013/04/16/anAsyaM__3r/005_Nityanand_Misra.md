I have trouble following the above. What is क्यडटू? सलोपश्र -> सलोपश्च?? ज्जिवात्?? What is क्तिशानचर्द्धत्तियग्रेटापू??  
  
दुर्मनायमानानाम् is from दुर्मनस् क्यङ् शानच् आम् I believe.  
  
दुर्मनस् -> भृशादिभ्यो भुव्यच्वेर्लोपश्च हलः (3-1-12) -> दुर्मनस् क्यङ् -> अनुबन्धलोपः, सलोपः -> दुर्मन य -> अकृत्सार्वधातुकयोर्दीर्घः (7-4-25) -> दुर्मना य -> दुर्मनाय -> सनाद्यन्ता धातवः (3-1-32) -> धातुसंज्ञा -> अनुदात्तङित आत्मनेपदम् (1-3-12) -> वर्त्तमाने लट् (3-2-123) -> दुर्मनाय लट् -> लटः शतृशानचावप्रथमासमानाधिकरणे (3-2-124) -> दुर्मनाय शानच् -> अनुबन्धलोपः -> दुर्मनाय आन -> तिङ्शित्सार्वधातुकम् (3-4-113)-> कर्तरि शप्‌ (3-1-68) -> दुर्मनाय शप् आन -> अनुबन्धलोपः -> दुर्मनाय अ आन -> अतो गुणे (6-1-97) -> पररूप -> दुर्मनाय आन -> आने मुँक् (7-2-82)-> मुँगागमः -> आद्यन्तौ टकितौ -> अन्तागमः -> दुर्मनाय मुँक् आन -> अनुबन्धलोपः -> दुर्मनाय म् आन -> दुर्मनायमान  
  
Further steps are simple प्रातिपदिक संज्ञा -> सुप्प्राप्तिः -> दुर्मनायमान आम् -> नुट् आगम -> दीर्घ -> दुर्मनायमानानाम्
