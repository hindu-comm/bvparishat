Salutation to sanskrit scholars,

  

We found a word वेश्य in a manuscript which refers to वैश्यवंशः.

we generally find the word वेश्या - with the root + suffixes- वेश् - यत् +टाप्

Similarly वैश्य - विश्+ क्विप्+ ष्यञ्

Can we derive वेश्य with the root विश् referring वैश्यवंशः.

If so with which suffix or प्रत्ययः?
