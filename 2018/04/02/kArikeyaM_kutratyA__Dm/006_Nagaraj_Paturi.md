\[अथापादानाधिकारः\]

निर्दिष्टविषयं किं चिद् उपात्तविषयं तथा /

अपॆक्षितक्रियं चॆति त्रिधापादानम् उच्यतॆ // वाक्य\_३,७।१३६ //

संयॊगभॆदाद् भिन्नात्मा गमिर् ऎव भ्रमिर् यथा /

ध्रुवावधिर् अपायॊ ऽपि समवॆतस् तथाध्रुवॆ // वाक्य\_३,७।१३७ //

द्रव्यस्वभावॊ न ध्रौव्यम् इति सूत्रॆ प्रतीयतॆ /

अपायविषयं ध्रौव्यं यत् तु तावद् विवक्षितम् // वाक्य\_३,७।१३८ //

सरणॆ दॆवदत्तस्य ध्रौव्यं पातॆ तु वाजिनः /

आविष्टं यद् अपायॆन तस्याध्रौव्यं प्रचक्षतॆ // वाक्य\_३,७।१३९ //

उभाव् अप्य् अध्रुवौ मॆषौ यद्य् अप्य् उभयकर्मजॆ /

विभागॆ प्रविभक्तॆ तु क्रियॆ तत्र विवक्षितॆ // वाक्य\_३,७।१४० //

मॆषान्तरक्रियापॆक्षम् अवधित्वं पृथक् पृथक् /

मॆषयॊः स्वक्रियापॆक्षं कर्तृत्वं च पृथक् पृथक् // वाक्य\_३,७।१४१ //

अभॆदॆन क्रियैका तु द्विसाध्या चॆद् विवक्षिता /

मॆषाव् अपायॆ कर्तारौ यद्य् अन्यॊ विद्यतॆ ऽवधिः // वाक्य\_३,७।१४२ //

गतिर् विना त्व् अवधिना नापाय इति गम्यतॆ /

वृक्षस्य पर्णं पततीत्य् ऎवं भाष्यॆ निदर्शितम् // वाक्य\_३,७।१४३ //

भॆदाभॆदौ पृथग्भावः स्थितिश् चॆति विरॊधिनः /

युगपन् न विवक्ष्यन्तॆ सर्वॆ धर्मा बलाहकॆ // वाक्य\_३,७।१४४ //

धनुषा विध्यतीत्य् अत्र विनापायविवक्षया /

करणत्वं यतॊ नास्ति तस्मात् तद् उभयं सह // वाक्य\_३,७।१४५ //

ऎकैव वा सती शक्तिर् द्विरूपा व्यवतिष्ठतॆ /

निमित्तं संज्ञयॊस् तत्र परया बाध्यतॆ ऽपरा // वाक्य\_३,७।१४६ //

निर्धारणॆ विभक्तॆ यॊ भीत्रादीनां च यॊ विधिः /

उपात्तापॆक्षितापायः सॊ ऽबुधप्रतिपत्तयॆ // वाक्य\_३,७।१४७ //

\[इत्य् अपादानाधिकारः\]

-----------------------------------------------------

  

अवधिः पञ्चम्यर्थः, "अपादाने पञ्चमी" (पाo सूo 2-3-29) इति सूत्रात्। तच्च "ध्रुवमपायेऽपादानम्" (पाo सूo 1-4-24) इति सूत्रात् अपायः=विश्लेषस्तज्जनकक्रिया, तत्रावधिभूतमपादानमित्यर्थकादवधिभूतमिति भावः।  
\<3-4-1>वाक्यपदीयदिशा अवधेः लक्षणम्  
  
उक्तञ्च वाक्यपदीये-  
  
अपाये यदुदासीनं चलं वा यदि वाऽचलम्।  
ध्रुवमेवातदावेशात्तदपादानमुच्यते।।  
पततो ध्रुव एवाश्वो यस्मादश्वात् पतत्यसौ।  
तस्याप्यश्वस्य पतने कुडयादि ध्रुवमुच्यते।।  
उभावप्यध्रुवौ मेषौ यद्यप्युभयकर्म्मजे (के)।  
विभागे प्रविभक्ते तु क्रिये तत्र विवक्षिते (व्यवस्थिते)।।  
मेषान्तरक्रियापेक्षमवधित्वं पृथक्‌ पृथक्।  
मेषयोः स्वक्रियापेक्षं कर्तृत्वञ्च पृथक् पृथक्।। इति।
