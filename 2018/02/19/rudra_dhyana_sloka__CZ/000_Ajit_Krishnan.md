तत्रभवतः सविनयं नमस्कृत्य,

  

I would like to request some help to understand the well-known rudra dhyaya sloka below.

  

आपातालनभःस्थलान्तभुवनब्रह्माण्डमाविस्फुरत्  
ज्योतिःस्फाटिकलिङ्गमौलिविलसत्पूर्णेन्दुवान्तामृतैः ।  
अस्तोकाप्लुतमेकमीशमनिशं रुद्रानुवाकाञ्जपन्  
ध्यायेदीप्सितसिद्धये ध्रुवपदं विप्रोऽभिषिञ्चेच्छिवम् ॥

  

In particular, I'd like to ask about the padavibhaga and vrittis for the first half.   

  

\(a\) Is the following पदविभागः correct? आपाताल+ब्रह्माण्डम् ; आविस्फुरत्+वान्तामृतैः

  

\(b\) Is आपाताल+ब्रह्माण्डं a बहुव्रीहिः samasa? Looking at some internet resources, some seem to link ब्रह्माण्डं with आविस्फुरत्, but that would have एकदेशान्वयः, and आविस्फुरत् is not णिजन्तम्.

  

\(c\) What is the vritti for ज्योतिःस्फाटिकलिङ्गं / how should it be understood?

  

Please excuse any childish mistakes above in this scholarly forum.

  

सस्नेहम्,
