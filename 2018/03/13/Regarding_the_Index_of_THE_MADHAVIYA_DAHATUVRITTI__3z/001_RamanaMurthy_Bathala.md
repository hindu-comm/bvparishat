Respected Peter Scharf,

Thanks for responding.

I checked the canonicalindex before posting the error itself. The canonicalindex gives correct sense निशाने. The issue is with the printed version of index.

Now it is a tricky situation.

1) To keep the integrity of the web page with respect to the original index in the printed text book, the sense/meaning बन्धने is to be kept as it is, without any changes.

2) If the purpose of the web page is to educate the viewers, the sense बन्धने has to be changed to निशाने. In this case, the integrity will be lost.

3) The third option is, without changing the sense, adding a foot note at the bottom of the web page, mentioning the printing mistake of the index in the printed book.



Regards

Ramana murthy
