In the Bhagavadgita 2nd ch. we have this phrase:
सशान्तिमाप्नोतिनकामकामी॥ This is the most exalted state that a human can attain. So, while expressing condolence one can modify the above to *स**शान्तिमाप्न्युयात्* or if we want the same phrase to be gender inclusive, can say: शान्तिमाप्न्युयादसौ. This 'shanti' is popular in Upanishads too. तमात्मस्थं येऽनुपश्यन्ति धीरास्तेषां शान्तिःशाश्वतीनेतरेषाम् kaThopanishat. Actually the RIP is not completely unfriendly to us. We can suitably modify that as per the above suggestion.

  

In Tamizh one can have: அவர் பேரின்பப் பெரு வாழ்வு பெருக. (avar pErinbap peru vaazhvu peruka). இறைவனடி சேர்க \[Let him attain the Lord's feet'\]
