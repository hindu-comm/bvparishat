।।हरये नमः।।  
।।विद्वद्भ्यो नमः।।  
\*न्यायमुक्तावल्यां काचिद् जिज्ञासा।\*

जातिनिरूपणे तद्बाधकसङ्ग्रहे षड्बाधकाः प्रकीर्तिताः।  
तत्र व्यक्तेरभेदः न सम्यक् अवधारितम्। आकाशत्वं जातिर्नास्ति इति निरूपीतमत्र। अस्य मदवधारितम् यद् अर्थं तद् वक्ष्ये आंग्लभाषायाम् इति - Aakash, Kaal etc are one and not many. So there cannot be an अनुगतप्रतीति like anugat pratiti in any dravya. Like when someone sees घट, पट, दण्ड etc, there is an अनुगतप्रतीति of \*"this is पृथ्वी। this is पृथ्वी। this is पृथ्वी।"\* in each of them.  
But Aakash, Kaal etc being only one, there is no anugat pratiti so it can't be a jati. Only such a brief description is not satisfactory. So asking great schollars about Lakshan.

\|\|Lakshan in dinkariya tika\|\|  
तस्य परिभाषेयं निरूप्यते दिनकरीयटीकया = \*"स्वाश्रयनिष्ठस्वाश्रयप्रतियोगिकभेदाभावो व्यक्त्यभेदः"\* इति।  
स्व - \_आकशत्वम्\_। तस्य आश्रयः - \_आकाशः\_। तस्मिन् निष्ठम् - \_आकाशत्वम्\_। \*स्वाश्रयनिष्ठे आकाशत्वे\*

स्वाश्रयः - \_आकाशः\_। तस्य प्रतियोगिक = \_आकाशान्योन्याभावः\_। प्रतियोगिकश्चासौ भेदः प्रतियोगिकभेदः - \_आकाशान्योन्याभावः यथा घट:।\_  
\*स्वाश्रयप्रतियोगिकभेदः - घट:।\* तस्य अभावः।

\*स्वाश्रयनिष्ठे आकाशत्वे, स्वाश्रयप्रतियोगिकभेदः - घट:, तस्य अभावः।\* आकाशत्वं घटो नास्ति इति तयोः भेदः दृश्यते।

किन्तु भेदस्य अभावः तु दृष्टो नहि। भेदो दृष्टोयम् । \*घटः आकाशो नास्ति\* इति भेदः स्पष्ट: दृश्यते तर्हि कथं भेदाभावः?

\*आकाशात्यन्ताभव:\* मन्यते वा? यथा \*\_स्वाश्रयनिष्ठे आकाशत्वे स्वाश्रयप्रतियोगिकभेदस्य घटस्य अत्यन्ताभावः?\_\*  
Kindly enlighten me.  
Regards

Rishi Goswami
