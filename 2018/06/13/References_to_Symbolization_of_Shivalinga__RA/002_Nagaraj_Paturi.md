Sir,

  

I don't have any reference for this.

  

I believe there can not be any because this view is wrong.

  

(Since the interpreatation brought for discussion itself has some explicit descriptions, response to that is compelled to have similar explicits. I may be forgiven for this.)

  

If the linga is male organ and is viewed as entered into female organ, and the top of the shivalinga is viewed as glans penis, isn't there a contradiction?

  

How can the glans penis be visible outside that too as the outer extreme?

  

Even if a reference is found , the claim in the reference needs to answer this question.

  

-------------------------------------------------------------------------------------------------------------------------------------------------------

  

In my view, the the design/form given to the structure that supports the s'ivalinga is keepingin view the function of abhisheka liquids flowing out and getting collected repeatedly as and when the structure fills fully so that space is created for more abhisheka fluids is created.A symbolic meaning. is being forced onto the form provided for this function.

  

Because this meaning is being forced, it is not able to answer questions such as the above.
